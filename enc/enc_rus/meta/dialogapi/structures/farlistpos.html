FarListPos

главная | Dialog API
Структура FarListPos задает параметры для позиционирования курсора в DI_LISTBOX и DI_COMBOBOX. 
struct FarListPos
{
  int SelectPos;
  int TopPos;
};

Элементы

SelectPos
Позиция текущего элемента.
TopPos
Позиция первого видимого элемента, -1 для автоматической подстройки под значение SelectPos.
