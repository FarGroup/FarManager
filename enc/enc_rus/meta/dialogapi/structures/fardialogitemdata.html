FarDialogItemData

главная | Dialog API
Структура FarDialogItemData описывает данные, передаваемые (получаемые) элементу диалога сообщением DM_SETTEXT (DM_GETTEXT) в качестве параметра Param2. 
struct FarDialogItemData
{
  int   PtrLength;
  char *PtrData;
};

Элементы

PtrLength
Размер данных в PtrData без завершающего символа конца строки '\0'.
PtrData
Указатель на передаваемые данные.
