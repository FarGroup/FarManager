FarListFind

главная | Dialog API
Структура FarListFind задает параметры для поиска элемента в DI_LISTBOX и DI_COMBOBOX. 
struct FarListFind
{
  int StartIndex;
  const char *Pattern;
  DWORD Flags;
  DWORD Reserved;
};

Элементы

StartIndex
Индекс элемента, с которого начинается поиск.
Pattern
Шаблон для поиска по правилам функции CmpName
Flags
Дополнительные флаги. Одно из следующих значений (перечисление FARLISTFINDFLAGS):
ФлагОписание
LIFIND_EXACTMATCH Поле Pattern не является шаблоном - рассматривать как обычную подстроку поиска.
Reserved
Резерв.
