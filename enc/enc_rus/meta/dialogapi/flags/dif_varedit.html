DIF_VAREDIT

главная | Dialog API | Флаги элементов диалога

Флаг DIF_VAREDIT - это "преодоление барьера в 512 байт для поля ввода". Менеджер диалогов при работе со строкой редактирования или комбинированным списком (с выставленным флагом DIF_VAREDIT) будет учитывать только члены Ptr.* структуры FarDialogItem.

Элементы

Флаг DIF_VAREDIT имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_COMBOBOX комбинированный список
DI_EDIT строка ввода

Замечания

  1. Плагин должен сам позаботится о выделении памяти для FarDialogItem.Ptr.PtrData и корректно заполнить размер этой памяти (FarDialogItem.Ptr.PtrLength).

  2. Если размер используемых данных не превышает значения 512 байт (или sizeof(FarDialogItem.Data)), то не имеет смысла использовать флаг DIF_VAREDIT.
