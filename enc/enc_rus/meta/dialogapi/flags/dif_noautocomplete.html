DIF_NOAUTOCOMPLETE

главная | флаги элементов диалога | Dialog API
Флаг DIF_NOAUTOCOMPLETE отключает автодополнение для строк редактирования.

Элементы

Флаг DIF_NOAUTOCOMPLETE имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_COMBOBOX комбинированный список
DI_EDIT строка ввода
DI_FIXEDIT поле ввода фиксированного размера
DI_PSWEDIT поле ввода пароля
