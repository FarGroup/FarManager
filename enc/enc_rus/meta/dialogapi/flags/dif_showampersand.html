DIF_SHOWAMPERSAND

главная | флаги элементов диалога | Dialog API
Флаг DIF_SHOWAMPERSAND заставляет показывать в текстовой строке и рамках амперсанд (&), не используя его для определения горячей клавиши.

Элементы

Флаг DIF_SHOWAMPERSAND имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_BUTTON кнопка (Push Button)
DI_CHECKBOX контрольный переключатель (Check Box)
DI_RADIOBUTTON селекторная кнопка (Radio Button)
DI_TEXT текстовая строка
DI_VTEXT вертикальная текстовая строка
DI_DOUBLEBOX двойная рамка
DI_SINGLEBOX одиночная рамка
