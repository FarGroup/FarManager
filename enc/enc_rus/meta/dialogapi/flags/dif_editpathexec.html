DIF_EDITPATHEXEC

главная | флаги элементов диалога | Dialog API
Флаг DIF_EDITPATHEXEC ограничивает автодополнение в строках ввода только именами исполнимых файлов (%PATHEXT%), которые находятся в каталогах, перечисленных в %PATH%.

Элементы

Флаг DIF_EDITPATHEXEC имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_COMBOBOX комбинированный список (без флага DIF_DROPDOWNLIST)
DI_EDIT строка ввода
