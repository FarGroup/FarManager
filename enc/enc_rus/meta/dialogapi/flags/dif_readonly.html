DIF_READONLY

главная | Dialog API | Флаги элементов диалога

Флаг DIF_READONLY переводит строки редактирования в состояние "только для чтения".

Элементы

Флаг DIF_READONLY имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_COMBOBOX комбинированный список
DI_EDIT строка ввода
DI_FIXEDIT поле ввода фиксированного размера
DI_PSWEDIT поле ввода пароля
