DIF_DISABLE

главная | Dialog API | Флаги элементов диалога

Флаг DIF_DISABLE указывает Менеджеру Диалогов, что данный элемент недоступен (Disabled). Это значит, что такой элемент диалога не получит фокуса ввода и не будет реагировать на мышь, но в тоже время элемент с этим флагом можно изменить программно.

Элементы

Флаг DIF_DISABLE имеет смысл для следующих элементов диалога:
ЭлементОписание
Все Все элементы диалога могут иметь состояние "недоступен" (Disabled)

Замечания

  1. Для изменения состояния Enable/Disable элемента пошлите сообщение DM_ENABLE Менеджеру Диалогов (функцией SendDlgMessage).
  2. Дизабленные редактируемые элементы диалога отображаются цветом COL_DIALOGEDITDISABLED. Если вы хотите изменить цвет элемента, то в обработчике диалога необходимо добавить, например, следующий код: 
    ...
if(msg == DN_CTLCOLORDLGITEM)
{
  if(Param1 >= 9 && Param1 <= 13)
  {
    int Lo=Info.AdvControl(Info.ModuleNumber,ACTL_GETCOLOR,(void*)COL_DIALOGEDIT);
    int Hi=Info.AdvControl(Info.ModuleNumber,ACTL_GETCOLOR,(void*)COL_DIALOGEDITSELECTED);
    return MAKELONG(MAKEWORD(Lo,Hi),MAKEWORD(Lo,Hi));
  }
}
...
    В данном примере меняются предопределенные цвета для дизабленных элементов на цвета, соответствующие нормальному состоянию элемента редактирования.
