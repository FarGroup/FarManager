DIF_USELASTHISTORY

главная | флаги элементов диалога | Dialog API
Строка ввода с установленными флагами DIF_HISTORY и DIF_USELASTHISTORY получает начальное значение из последнего элемента истории, если начальное значение поля редактирования не задано.

Элементы

Флаг DIF_USELASTHISTORY имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_EDIT строка ввода
DI_FIXEDIT поле ввода фиксированного размера
