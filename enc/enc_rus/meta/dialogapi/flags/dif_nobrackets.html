DIF_NOBRACKETS

главная | Dialog API | Флаги элементов диалога

Флаг DIF_NOBRACKETS заставляет Менеджер Диалогов показывать заголовок кнопки DI_BUTTON без квадратных скобок.

Элементы

Флаг DIF_NOBRACKETS имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_BUTTON кнопка (Push Button)
