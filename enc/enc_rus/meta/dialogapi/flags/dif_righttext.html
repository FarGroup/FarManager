DIF_RIGHTTEXT

главная | флаги элементов диалога | Dialog API
Флаг DIF_RIGHTTEXT позволяет выравнивать текст элемента или заголовок рамки по правому краю (нижнему краю для DI_VTEXT).

Элементы

Флаг DIF_RIGHTTEXT имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_TEXT текстовая строка
DI_VTEXT вертикальная текстовая строка
DI_DOUBLEBOX двойная рамка
DI_SINGLEBOX одиночная рамка

Замечания

При указании этого флага, плагин должен обязательно корректно заполнить координаты X1,X2 и Y1,Y2 (для DI_VTEXT).
