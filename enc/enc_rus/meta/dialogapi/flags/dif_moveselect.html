DIF_MOVESELECT

главная | Dialog API | Флаги элементов диалога

Флаг DIF_MOVESELECT позволяет изменять выбранный элемент внутри группы элементов DI_RADIOBUTTON при изменении фокуса ввода. При этом сам элемент будет иметь другой вид - отрисовка без круглых скобок. Пример работы этого флага см. в диалоге настройки цветов FAR Manager.

Элементы

Флаг DIF_MOVESELECT имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_RADIOBUTTON селекторная кнопка (Radio Button)
