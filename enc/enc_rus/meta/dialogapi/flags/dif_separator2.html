DIF_SEPARATOR2

главная | флаги элементов диалога | Dialog API
Текстовая строка DI_TEXT с флагом DIF_SEPARATOR2 отображается как двойная горизонтальная линия с двойными боковыми линиями.

Элементы

Флаг DIF_SEPARATOR2 имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_TEXT текстовая строка
DI_VTEXT вертикальная текстовая строка

Замечания

  1. Координаты отрисовки разделительной линии (левый край диалога = 0): X1 = X1_Диалога + N; X2 = X2_Диалога - N,
    где N равно 3 или 0 (при вызове диалога с флагом FDLG_SMALLDIALOG).
  2. Координата X1 влияет только на позиционирование текстовой строки (отображается поверх разделительной линии) из Data. Если вам нужна разделительная линия с центрированным текстом, то укажите X1 = -1, иначе текст будет отрисован с координаты X1.
  3. Если Data содержит пустую строку, элемент отображается просто как разделительная линия.
