DIF_SEPARATOR

главная | Dialog API | Флаги элементов диалога

Текстовая строка DI_TEXT с флагом DIF_SEPARATOR отображается как одинарная горизонтальная линия с двойными боковыми линиями.

Элементы

Флаг DIF_SEPARATOR имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_TEXT Текстовая строка.

Замечания

  1. Координаты отрисовки разделительной линии (левый край диалога = 0): X1 = X1_Диалога + 3; X2 = X2_Диалога - 3.
  2. Координата X1 влияет только на позиционирование текстовой строки (отображается поверх разделительной линии) из Data. Если вам нужна разделительная линия с центрированным текстом, то укажите X1 = -1.
  3. Если Data содержит пустую строку, элемент отображается просто как разделительная линия.
