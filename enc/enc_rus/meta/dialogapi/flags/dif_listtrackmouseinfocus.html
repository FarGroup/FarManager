DIF_LISTTRACKMOUSEINFOCUS

главная | флаги элементов диалога | Dialog API
Флаг DIF_LISTTRACKMOUSEINFOCUS позволяет задать реакцию на движение мыши в списке только если элемент находится в фокусе. Обычное поведение списка - не реагировать на движение мыши (не перемещать курсор в списке).

Элементы

Флаг DIF_LISTTRACKMOUSEINFOCUS имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_LISTBOX список
