DIF_EDITOR

главная | флаги элементов диалога | Dialog API
Последовательно определённые поля ввода (DI_EDIT), имеющие флаг DIF_EDITOR группируются в редактор с возможностью вставки и удаления строк.

Элементы

Флаг DIF_EDITOR имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_EDIT строка ввода

Замечания

Для того, чтобы в диалоге создать область редактирования размером в 5 строк, необходимо разместить 5 элементов редактирования типа DI_EDIT с установленным флагом DIF_EDITOR, как показано в пример: 
DI_EDIT,5,3,29,3,1,1,0,0,DIF_EDITOR,0,L"",0,0,
DI_EDIT,5,4,29,4,0,1,0,0,DIF_EDITOR,0,L"",0,0,
DI_EDIT,5,5,29,5,0,1,0,0,DIF_EDITOR,0,L"",0,0,
DI_EDIT,5,6,29,6,0,1,0,0,DIF_EDITOR,0,L"",0,0,
DI_EDIT,5,7,29,7,0,1,0,0,DIF_EDITOR,0,L"",0,0
