DIF_GROUP

главная | флаги элементов диалога | Dialog API
Флаг DIF_GROUP выставленный у первого элемента DI_RADIOBUTTON группирует последовательно объявленные радиокнопки: 
// диалог выбора цветов в Far Manager:
...
// первая группа радиокнопок
DI_RADIOBUTTON,6,3,0,0,0,0,F_LIGHTGRAY|B_BLACK|DIF_GROUP|DIF_MOVESELECT,0,L"",
DI_RADIOBUTTON,6,4,0,0,0,0,F_BLACK|B_RED|DIF_MOVESELECT,0,L"",
...
// вторая группа радиокнопок
DI_RADIOBUTTON,21,3,0,0,0,0,F_LIGHTGRAY|B_BLACK|DIF_GROUP|DIF_MOVESELECT,0,L"",
DI_RADIOBUTTON,21,4,0,0,0,0,F_BLACK|B_RED|DIF_MOVESELECT,0,L"",
...

Элементы

Флаг DIF_GROUP имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_RADIOBUTTON селекторная кнопка (Radio Button)

Замечания

Укажите этот флаг для первого элемента в группе.
