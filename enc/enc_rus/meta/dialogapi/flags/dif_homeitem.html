DIF_HOMEITEM

главная | флаги элементов диалога | Dialog API
Первый элемент управления с флагом DIF_HOMEITEM, который может получать клавиатурный фокус ввода, будет получать фокус, когда пользователь нажимает клавишу Home.

Элементы

Флаг DIF_HOMEITEM имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_BUTTON кнопка (Push Button)
DI_CHECKBOX контрольный переключатель (Check Box)
DI_EDIT строка ввода
DI_FIXEDIT поле ввода фиксированного размера
DI_COMBOBOX комбинированный список
DI_RADIOBUTTON селекторная кнопка (Radio Button)
DI_PSWEDIT поле ввода пароля
DI_LISTBOX список
DI_MEMOEDIT
DI_USERCONTROL элемент управления, определяемый программистом

Замечания

  • Элемент управления может получать клавиатурный фокус ввода, если он не имеет флагов DIF_NOFOCUS, DIF_DISABLE или DIF_HIDDEN.
  • Если более одного элемента диалога имеют флаг DIF_HOMEITEM, клавиша Home устанавливает фокус на первый из этих элементов, который может получать клавиатурный фокус ввода.
  • Если ни один из элементов диалога не имеет флага DIF_HOMEITEM, клавиша Home устанавливает фокус на первый элемент, который может получать клавиатурный фокус ввода.
  • Если элемент с флагом DIF_FOCUS не является первым элементом диалога, рекомендуется пометить этот элемент также и флагом DIF_HOMEITEM, чтобы клавиша Home устанавливала фокус на элемент, который получает фокус при показе диалога на экране.
