DN_DROPDOWNOPENED

главная | события | Dialog API
Событие DN_DROPDOWNOPENED посылается перед открытием Combobox или History и после закрытия.

Param1

ID элемента диалога, для которого будет открыт/закрыт список.

Param2

0 = закрыт, 1 = открыт.

Return

Не используется.
