DM_LISTGETDATA

главная | сообщения | Dialog API
Сообщение DM_LISTGETDATA посылается Менеджеру диалогов для получения данных, ассоциированных с элементом списка DI_COMBOBOX или DI_LISTBOX.

Param1

ID элемента диалога.

Param2

Индекс элемента списка.

Return

Данные, которые плагин ассоциировал с элементом списка с помощью сообщения DM_LISTSETDATA или NULL, если нет данных, ассоциированных с указанным пунктом списка. Более подробно о логике хранения данных см. замечание для структуры FarListItemData.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список
