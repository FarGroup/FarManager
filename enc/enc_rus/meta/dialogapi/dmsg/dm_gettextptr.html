DM_GETTEXTPTR

Сообщения | Dialog API
Сообщение DM_GETTEXTPTR посылается Менеджеру диалогов для получения содержимого строк ввода или заголовков элементов диалога. В отличие от сообщения DM_GETTEXT, это сообщение работает с указателем на строку.

Param1

ID требуемого элемента

Param2

Указатель на строку приемник.
Если Param2 = NULL - это еще один способ получить размер данных (см. DM_GETTEXTLENGTH)

Return

Размер данных без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Для строк ввода - содержимое строки, для прочих элементов - заголовки. Все то, что содержится в FarDialogItem.Data или FarDialogItem.PtrData.

Замечания

Сообщение DM_GETTEXTPTR в версиях FAR вплоть до 1.70 beta 3 возвращало размер данных с учетом символа конца строки.
