DM_SETINPUTNOTIFY

главная | сообщения | Dialog API
Сообщение DM_SETINPUTNOTIFY позволяет управлять механизмом посылки первичного необработанного события консоли (DN_INPUT) в диалоговую процедуру.

Param1

  • 0 - выключить
  • 1 - включить
  • -1 - получить состояние

Param2

0

Return

Предыдущее состояние.

Элементы

ЭлементОписание
Диалог сообщение затрагивает только диалог
