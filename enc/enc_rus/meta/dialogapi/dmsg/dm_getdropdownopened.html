DM_GETDROPDOWNOPENED

Сообщения | Dialog API
Сообщение DM_GETDROPDOWNOPENED служит для программного определения: "есть ли в диалоге раскрытый комбобокс или история".

Param1

0

Param2

0

Return

FALSE - в диалоге нет открытого комбобокса или истории.
TRUE - в диалоге есть открытый комбобокс или история.

Элементы

ЭлементОписание
Диалог сообщение затрагивает только диалог
