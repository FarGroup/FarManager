DN_KEY

События | Dialog API
Обработчик диалога получает событие DN_KEY, когда пользователь нажал клавишу.

Param1

ID элемента (обычно элемент, имеющий фокус ввода).

Param2

Внутренний код клавиши.

Return

TRUE - клавишу обработали сами.
FALSE - пусть обрабатывает клавишу стандартный внутренний обработчик диалога

Замечания

Пример

Пример обработки сообщения: 
// Центрировать диалог, если пользователь нажал Ctrl-Apps
case DN_KEY:
  if(Param2 == KEY_CTRLAPPS)
  {
    COORD c={-1,-1};
    Info.SendDlgMessage(hDlg,DM_MOVEDIALOG,TRUE,(long)&c);
    return TRUE;
  }
