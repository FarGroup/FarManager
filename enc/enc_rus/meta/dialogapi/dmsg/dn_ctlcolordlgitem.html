DN_CTLCOLORDLGITEM

События | Dialog API
Событие DN_CTLCOLORDLGITEM посылается процедуре обработки диалога перед отрисовкой конкретного элемента диалога.

Param1

ID элемента, который будет отрисован.

Param2

Атрибуты (цвет_фона+цвет_текста), с использованием которых обработчик диалога хочет отрисовать элемент:
DI_SINGLEBOX
DI_DOUBLEBOX
LO(LO=Title,HI=HiText), HI(LO=Box,0)
DI_TEXT
LO(LO=Title,HI=HiText), HI(LO=Box,0)
Цвет для рамки (Box) определен только для флагов DIF_SEPARATOR и DIF_SEPARATOR2.
DI_VTEXT
LO(LO=Title,HI=0), HI(0,0)
DI_EDIT
DI_FIXEDIT
DI_PSWEDIT
DI_COMBOBOX
LO(LO=EditLine,HI=SelectText), HI(LO=UnchangedColor, HI=History&ComboBox указатель)
DI_CHECKBOX
DI_RADIOBUTTON
LO(LO=Title,HI=HiText), HI(0,0)
DI_BUTTON
LO(LO=Title,HI=HiText), HI(0,0)
DI_LISTBOX
Для элемента DI_LISTBOX приходит только событие DN_CTLCOLORDLGLIST.

Return

Атрибуты (цвет_фона+цвет_текста), с использованием которых будет отрисован элемент.

Элементы

ЭлементОписание
Все Все элементы диалога, кроме DI_LISTBOX и DI_USERCONTROL

Замечания

В ответ на это сообщение процедура обработки диалога может установить свои атрибуты (цвет текста и фона) для заданного элемента.

Пример

Пример того, как диалог назначения макроса на клавишу в FAR Manager меняет цвет поля редактирования: 
/* 02 */ DI_EDIT,5,3,28,3,1,0,0,1,"",

case DN_CTLCOLORDLGITEM:
  // Unchanged у нас сидит в младшем байте старшего слова
  Param2&=0xFF00FFFFU;
  Param2|=(Param2&0xFF)<<16;
  return Param2;
