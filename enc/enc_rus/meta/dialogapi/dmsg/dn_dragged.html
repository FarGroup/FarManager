DN_DRAGGED

События | Dialog API
Событие DN_DRAGGED приходит в обработчик диалога как сигнал о начале и конце перемещения диалога.

Начало перемещения диалога

Param1

0

Param2

0

Return

TRUE - разрешить перемещение
FALSE - запретить перемещение

Конец перемещения диалога

Param1

1

Param2

0 - успешное перемещение
1 - отменили перемещение диалога, нажав Esc или правую клавишу мыши.

Return

Не имеет значения.

Замечания

  1. Событие DN_DRAGGED приходит до прорисовки диалога.
