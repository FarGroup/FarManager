DM_LISTSORT

Сообщения | Dialog API
Сообщение DM_LISTSORT посылается Менеджеру диалогов для сортировки элементов списка DI_COMBOBOX или DI_LISTBOX.

Param1

ID требуемого элемента диалога.

Param2

Направление сортировки: 0 - по возрастанию, 1 - по убыванию.

Return

TRUE - список отсортирован.
FALSE - ошибка при сортировке списка.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список

Замечания

Пример

// Сортировать список по убыванию (от 'Z' до 'A')
Info.SendDlgMessage(hDlg,DM_LISTSORT,ID,1);
