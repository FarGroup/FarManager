DM_SETCOMBOBOXEVENT

Сообщения | Dialog API
Сообщение DM_SETCOMBOBOXEVENT позволяет управлять механизмом посылки сообщений DN_KEY и DN_MOUSEEVENT в диалоговую процедуру при навигации в открытом элементе DI_COMBOBOX.

Param1

ID требуемого элемента DI_COMBOBOX.

Param2

Param2 комбинация следующих значений (перечисление FARCOMBOBOXEVENTTYPE):
ФлагОписание
CBET_KEY Включить приход событий DN_KEY
CBET_MOUSE Включить приход событий DN_MOUSEEVENT

Return

Предыдущее состояние.

Элементы

ЭлементОписание
DI_COMBOBOX комбинированный список

Замечания

По умолчанию при открытом DI_COMBOBOX события DN_KEY и DN_MOUSEEVENT в диалоговую процедуру не приходят.
