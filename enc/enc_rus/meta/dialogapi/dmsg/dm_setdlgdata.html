DM_SETDLGDATA

главная | сообщения | Dialog API
Сообщение DM_SETDLGDATA устанавливает указатель на пользовательские данные, связанные с диалогом. С каждым экземпляром диалога можно связать свои данные, предназначенные для использования плагином в его собственных целях.

Param1

0

Param2

Указатель на данные.

Return

Предыдущее значение.

Замечания

Первоначально здесь находится то, что передано в параметре Param функции DialogInit.
