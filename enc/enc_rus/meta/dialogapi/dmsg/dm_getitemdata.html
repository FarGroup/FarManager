DM_GETITEMDATA

главная | сообщения | Dialog API
Сообщение DM_GETITEMDATA возвращает указатель на связанные с элементом диалогом данные. Каждый элемент диалога может иметь свои данные, предназначенные для использования плагином, который создавал диалог.

Param1

ID элемента диалога.

Param2

0

Return

Менеджер Диалогов возвращает указатель на ранее переданные данные плагином (или 0, если плагин не ассоциировал данные с этим элементом диалога).

Элементы

ЭлементОписание
Все все элементы диалога
