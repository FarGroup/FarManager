DM_LISTADDSTR

главная | сообщения | Dialog API
Сообщение DM_LISTADDSTR посылается Менеджеру диалогов для добавления строки в список DI_COMBOBOX или DI_LISTBOX.

Param1

ID элемента диалога.

Param2

Указатель на добавляемую строку.

Return

  • Индекс добавленной строки в списке.
  • -1 в случае, если диалог не существует.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список
