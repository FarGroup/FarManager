DM_LISTINFO

главная | сообщения | Dialog API
Сообщение DM_LISTINFO позволяет получить информацию о списке DI_COMBOBOX или DI_LISTBOX.

Param1

ID элемента диалога.

Param2

Указатель на структуру FarListInfo.

Return

TRUE - структура FarListInfo заполнена.
FALSE - ошибка, в качестве Param2 передали значение NULL.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список

Замечания

Пример

// Получить информацию о списке
struct FarListInfo ListInfo={sizeof(FarListInfo)};
Info.SendDlgMessage(hDlg,DM_LISTINFO,ID,&ListInfo);
