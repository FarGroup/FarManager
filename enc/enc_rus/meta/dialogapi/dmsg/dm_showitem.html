DM_SHOWITEM

главная | сообщения | Dialog API
Сообщение DM_SHOWITEM посылается Менеджеру диалогов для того, чтобы показать или спрятать элемент диалога.

Param1

ID элемента диалога.

Param2

Одно из трёх значений:
  • -1 - получить состояние элемента
  • 0 - погасить элемент
  • 1 - показать элемент

Return

Предыдущее состояние элемента диалога.

Элементы

ЭлементОписание
Все Все элементы диалога

Замечания

  1. Элемент диалога принудительно перерисовывается, если он видим и Param2 = 1.
  2. Элемент диалога не перерисовывается для случая Param2 = -1.
