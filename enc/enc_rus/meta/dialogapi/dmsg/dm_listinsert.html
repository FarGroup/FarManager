DM_LISTINSERT

главная | сообщения | Dialog API
Сообщение DM_LISTINSERT посылается Менеджеру диалогов для вставки элемента в список DI_COMBOBOX или DI_LISTBOX.

Param1

ID элемента диалога.

Param2

Указатель на добавляемый пункт списка, структуру типа FarListInsert.

Return

Новое количество пунктов в списке.
-1 - ошибка при вставке данных.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список
