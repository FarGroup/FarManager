DM_GETTEXT

Сообщения | Dialog API
Сообщение DM_GETTEXT посылается Менеджеру диалогов для получения содержимого строк ввода или заголовков элементов диалога.

Param1

ID требуемого элемента

Param2

Указатель на структуру FarDialogItemData.
Если Param2 = NULL - это еще один способ получить требуемый размер данных (см. DM_GETTEXTLENGTH): 
LenData=SendDlgMessage(hDlg,DM_GETTEXT,1,(long)NULL);
эквивалентно 
LenData=SendDlgMessage(hDlg,DM_GETTEXTLENGTH,1,0);

Return

Размер данных без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Для строк ввода - содержимое строки, для прочих элементов - заголовки. Все то, что содержится в FarDialogItem.Data или FarDialogItem.PtrData.

Замечания

Сообщение DM_GETTEXT в версиях FAR вплоть до 1.70 beta 3 возвращало размер данных с учетом символа конца строки.
