DM_SETTEXTPTR

Сообщения | Dialog API
Сообщение DM_SETTEXTPTR посылается Менеджеру диалогов для установки содержимого строк ввода или заголовков элементов диалога в новое значение. В отличии от сообщения DM_SETTEXT, это сообщение работает с указателем на строку.

Param1

ID требуемого элемента диалога.

Param2

Адрес строки, содержащей текст. Значение этого параметра равное NULL игнорируется.

Return

Размер установленных данных без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Все элементы диалога

Замечания

  1. Для DI_COMBOBOX влияет только на строку редактирования (не распространяется на DIF_DROPDOWNLIST)
  2. Сообщение DM_SETTEXTPTR в версиях FAR вплоть до 1.70 beta 3 возвращало размер данных с учетом символа конца строки.
