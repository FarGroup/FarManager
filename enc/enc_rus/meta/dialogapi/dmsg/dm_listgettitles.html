DM_LISTGETTITLES

главная | сообщения | Dialog API
Сообщение DM_LISTGETTITLES посылается Менеджеру диалогов для получения заголовков (верхнего и нижнего) у элемента DI_LISTBOX.

Param1

ID элемента диалога.

Param2

Указатель на структуру типа FarListTitles, в которую будут помещены заголовки.

Return

TRUE - заголовки получены
FALSE -элемент не является списком

Элементы

ЭлементОписание
DI_LISTBOX список

Замечания

Для получения заголовков необходимо предварительно заполнить структуру FarListTitles: 
wchar_t Title[100];
wchar_t Bottom[100];
FarListTitles ListTitle={sizeof(FarListTitles)};
ListTitle.Title=Title;
ListTitle.TitleSize=100;
ListTitle.Bottom=Bottom;
ListTitle.BottomSize=100;
Info.SendDlgMessage(hDlg,DM_LISTGETTITLES,ID,&ListTitle);
