DM_GETDIALOGINFO

главная | события | Dialog API
Команда DM_GETDIALOGINFO посылается обработчику диалога, для запроса информации о диалоге Far Manager'а или другого плагина.

Param1

0

Param2

Указатель на структуру DialogInfo, получающую данные. Плагин должен корректно заполнить поле DialogInfo.StructSize перед вызовом.

Return

TRUE - если структура заполнена.
FALSE - соощение проигнорировано.
