DM_KEY

главная | сообщения | Dialog API
Используя сообщение DM_KEY плагин передаёт массив клавиш Менеджеру диалогов.

Param1

Количество передаваемых клавиш.

Param2

Массив структур типа INPUT_RECORD.

Return

0

Замечания

Обработчик диалога не получит события DN_CONTROLINPUT в ответ на посылку Менеджеру диалогов сообщения DM_KEY.

Пример

Смотрите также:
DialogInit, DN_CONTROLINPUT
