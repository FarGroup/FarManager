DM_KEY

Сообщения | Dialog API
Используя сообщение DM_KEY плагин передает массив значений кодов клавиш Менеджеру диалогов.

Param1

Количество передаваемых клавиш.

Param2

Указатель на массив (типа DWORD) внутренних кодов клавиш.

Return

0

Замечания

Обработчик диалога не получит события DN_KEY в ответ на посылку Менеджеру диалогов сообщения DM_KEY.

Пример

Пример передачи набора клавиш диалогу (фокусному элементу): 

// в конец строки редактирования добавить точку с запятой.
DWORD Keys[2]={KEY_END,';'};

Info.SendDlgMessage(hDlg,DM_KEY,sizeof(Keys)/sizeof(Keys[0]),(long)Keys);
