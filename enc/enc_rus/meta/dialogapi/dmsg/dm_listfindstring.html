DM_LISTFINDSTRING

главная | сообщения | Dialog API
Сообщение DM_LISTFINDSTRING посылается Менеджеру диалогов для поиска элемента списка DI_COMBOBOX или DI_LISTBOX по шаблону.

Param1

ID элемента диалога.

Param2

Указатель на структуру, описывающую искомый пункт списка, структуру типа FarListFind.

Return

Позиция найденного элемента в списке (начиная с нуля).
-1 - элемент в списке не найден.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список
