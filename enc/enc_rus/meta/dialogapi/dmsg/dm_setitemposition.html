DM_SETITEMPOSITION

главная | сообщения | Dialog API
Сообщение DM_SETITEMPOSITION посылается Менеджеру диалогов для изменения позиции элемента.

Param1

ID элемента диалога.

Param2

Указатель на структуру SMALL_RECT с новыми координатами.

Return

TRUE - позиция элемента изменена.
FALSE - если такого ID элемента не существует или координаты заданы неверно (Left > Right или Top > Bottom)
