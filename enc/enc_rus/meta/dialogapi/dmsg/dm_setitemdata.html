DM_SETITEMDATA

главная | сообщения | Dialog API
Сообщение DM_SETITEMDATA устанавливает указатель на пользовательские данные, связанные с элементом диалога. С каждым элементом диалога можно связать свои данные, предназначенные для использования плагином в его собственных целях.

Param1

ID элемента управления.

Param2

Указатель на данные.

Return

Предыдущее значение.
