DM_LISTGETCURPOS

главная | сообщения | Dialog API
Сообщение DM_LISTGETCURPOS посылается Менеджеру диалогов для получения текущей позиции курсора в списке DI_COMBOBOX или DI_LISTBOX.

Param1

ID элемента диалога.

Param2

Указатель на структуру FarListPos, в которую будет записана дополнительная информация о позиции в списке, или NULL, если дополнительная информация не требуется.

Return

Текущая позиция курсора в списке.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список

Замечания

Пример

// получить текущую позицию

CurPos=Info.SendDlgMessage(hDlg,DM_LISTGETCURPOS,ID,NULL);
