DM_GETDLGITEMSHORT

главная | сообщения | Dialog API
Сообщение DM_GETDLGITEMSHORT посылается Менеджеру диалогов для получения информации о заданном элементе. Отличается от DM_GETDLGITEM тем, что FarDialogItem.Data, FarDialogItem.Mask и FarDialogItem.History всегда возвращаются как NULL.

Param1

ID элемента диалога.

Param2

Указатель на структуру FarDialogItem.

Return

TRUE - данные скопированы или FALSE - для неверного ID элемента.

Элементы

ЭлементОписание
Все все элементы диалога

Замечания

Пример

// получить 12-й элемент
struct FarDialogItem DialogItem;
Info.SendDlgMessage(hDlg,DM_GETDLGITEMSHORT,12,&DialogItem);
