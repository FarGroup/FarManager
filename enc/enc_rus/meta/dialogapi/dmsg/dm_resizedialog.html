DM_RESIZEDIALOG

главная | сообщения | Dialog API
Сообщение DM_RESIZEDIALOG посылается менеджеру диалогов для изменения размеров диалога.

Param1

0

Param2

Указатель на структуру COORD, в которой указаны новые размеры диалогового окна.

Return

Структура COORD, содержащая новые размеры диалогового окна.

Замечания

Если Far Manager был запущен с параметром "far.exe /w", то размеры окна могут отличаться от размеров экранного буфера Far Manager и Ваш диалог может "уплыть" за границы экрана, учитывайте этот случай в своем плагине.
