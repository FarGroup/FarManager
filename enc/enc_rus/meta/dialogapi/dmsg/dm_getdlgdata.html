DM_GETDLGDATA

главная | сообщения | Dialog API
Сообщение DM_GETDLGDATA возвращает указатель на связанные с диалогом данные. Каждый экземпляр диалога может иметь свои данные, предназначенные для использования плагином, который создавал диалог.

Param1

0

Param2

0

Return

Менеджер Диалогов возвращает указатель на ранее переданные данные плагином.

Замечания

Первоначально здесь находится то, что передано в аргументе Param функции DialogInit.
