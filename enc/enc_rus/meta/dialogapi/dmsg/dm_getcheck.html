DM_GETCHECK

главная | сообщения | Dialog API
Сообщение DM_GETCHECK посылается Менеджеру диалогов для получения состояния элементов DI_CHECKBOX и DI_RADIOBUTTON.

Param1

ID требуемого элемента.

Param2

0

Return

Состояние элемента: BSTATE_UNCHECKED, BSTATE_CHECKED, BSTATE_3STATE.

Элементы

ЭлементОписание
DI_CHECKBOX контрольный переключатель (Check Box)
DI_RADIOBUTTON селекторная кнопка (Radio Button)
