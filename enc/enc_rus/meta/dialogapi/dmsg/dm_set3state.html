DM_SET3STATE

Сообщения | Dialog API
Сообщение DM_SET3STATE посылается Менеджеру диалогов для модификации стиля элемента DI_CHECKBOX.

Param1

ID требуемого элемента

Param2

  • TRUE - контрольный переключатель может иметь 3 состояния.
  • FALSE - контрольный переключатель может иметь 2 состояния.

Return

Предыдущий стиль управляющего элемента.

Элементы

ЭлементОписание
DI_CHECKBOX контрольный переключатель (Check Box).
