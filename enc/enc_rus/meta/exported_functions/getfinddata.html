GetFindData

главная | экспортируемые функции
Функция GetFindData вызывается для получения списка файлов из текущего каталога эмулируемой файловой системы
(FAR плагину: "дай-ка я взгляну на списочек твоих файлов, память сам выдели :-)"). 
int WINAPI GetFindData(
  HANDLE hPlugin,
  struct PluginPanelItem **pPanelItem,
  int *pItemsNumber,
  int OpMode
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPlugin или OpenFilePlugin.
pPanelItem
Указывает на переменную, которая получит адрес массива структур PluginPanelItem.
pItemsNumber
Указывает на переменную, которая получит число структур PluginPanelItem.
OpMode
Комбинация флагов operation mode. Эта функция должна быть готова обработать флаг OPM_FIND.

Возвращаемое значение

В случае успеха должно возвращаться TRUE. Иначе FALSE.

Замечания

  1. Плагин сам должен выделять память под запрошенные данные. Если это происходит, то необходимо позаботиться об освобождении памяти, т.е. не забыть бы про экспортирование функции FreeFindData
  2. Если функция вернет FALSE, плагин будет закрыт.
  3. В некоторых случаях (пример: поиск в архивах по Alt-F7) физически панель плагина не создается, поэтому необходимо обязательно проверять код возврата у Control, чтобы не упасть в самый не подходящий момент, выполнив работу для панели, которой нет.
  4. Если вы не хотите, чтобы ваш плагин использовался во время "поиска в архивах" ("[x] Искать в архивах" в диалоге поиска), то возвращайте FALSE в случае, если OpMode содержит флаг OPM_FIND.
