Compare

главная | экспортируемые функции
Плагин может экспортировать функцию Compare, чтобы перекрыть используемый по умолчанию алгоритм сортировки файлов. 
int WINAPI Compare(
  HANDLE hPlugin,
  const struct PluginPanelItem *Item1,
  const struct PluginPanelItem *Item2,
  unsigned int Mode
);

Параметры

hPlugin
Описатель плагина, возвращаемый функциями OpenPlugin или OpenFilePlugin.
Item1, Item2
Указатели на структуру PluginPanelItem для сравнения.
Mode
См. Методы сортировки

Возвращаемое значение

Эта функция возвращает одно из следующих значение типа int:

-1 если Item1 < Item2
 0 если Item1 == Item2
 1 если Item1 > Item2
-2 если используется внутренняя функция Compare для данного метода сортировки.

Замечания

Для сортировки элементов FAR использует стандартную RTL-функцию qsort, которая реализует нестабильную сортировку. Иными словами, если элементы массива равны с точки зрения функции сравнения, на панели они будут отображаться в произвольном порядке, меняющемся при каждом перечитывании панели.
