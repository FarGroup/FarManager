ProcessHostFile

главная | экспортируемые функции | поддержка архивов
Функция ProcessHostFile вызывается, чтобы выполнить архивные команды FAR Manager. Рекомендуется поместить сюда дополнительные операции, которые обрабатывают файл, на основании которого плагин эмулирует файловую систему. 
int WINAPI ProcessHostFile(
  HANDLE hPlugin,
  struct PluginPanelItem *PanelItem,
  int ItemsNumber,
  int OpMode
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPlugin или OpenFilePlugin.
PanelItem
Указывает на массив структур PluginPanelItem. Каждая структура соответствует файлу, выбранному в панели плагина.
ItemsNumber
Количество структур PluginPanelItem в массиве PanelItem.
OpMode
Комбинация флагов operation mode. Для этой функции равна 0 или OPM_TOPLEVEL.

Возвращаемое значение

В случае успеха должно возвращаться TRUE. Иначе - FALSE.

Примечание

Если операция не удалась, но часть файлов была успешно обработана, плагин может удалить выделение только с обработанных файлов. Для этого он должен очистить флаг PPIF_SELECTED у обработанных элементов переданного функции списка PluginPanelItem.
