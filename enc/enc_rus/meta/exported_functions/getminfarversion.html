GetMinFarVersion

главная | экспортируемые функции
FAR вызывает функцию GetMinFarVersion для получения информации о минимальной версии, с которой согласен работать плагин.
int WINAPI GetMinFarVersion(void);

Параметры

Нет.

Возвращаемое значение

Возвращаемое значение должно быть в формате 0xZZZZXXYY:
КомпонентаОписание
ZZZZ номер билда (FAR 1.70.387 = 0x0183)
XX major version (FAR 1.70 = 0x01)
YY minor version (FAR 1.70 = 0x46)

Рекомендуется использовать макрос MAKEFARVERSION.

Замечания

Если возвращаемое функцией значение больше чем текущая версия FAR Manager (включая номер билда) - выводится сообщение об ошибке и плагин выгружается (потом он пропадает из меню плагинов).

Пример

// Укажем, что этот плагин требует 591-й билд FAR Manager 1.70
int WINAPI _export GetMinFarVersion(void)
{
  return MAKEFARVERSION(1,70,591);
}

// А этот плагин будет гарантировано работать FAR Manager начиная
// с версии 1.70 и выше.
int WINAPI _export GetMinFarVersion(void)
{
  return MAKEFARVERSION(1,70,0);
}
Смотрите также:
MAKEFARVERSION, FARMANAGERVERSION, "Плагин для разных версий FAR Manager"
