FreeVirtualFindData

главная | экспортируемые функции
Функция FreeVirtualFindData должна освободить память, выделенную функцией GetVirtualFindData.
void WINAPI FreeVirtualFindData(
  HANDLE hPlugin,
  struct PluginPanelItem *PanelItem,
  int ItemsNumber
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPlugin или OpenFilePlugin.
PanelItem
Указывает на массив структур PluginPanelItem, ранее выделенный функцией GetVirtualFindData.
ItemsNumber
Количество структур PluginPanelItem в массиве PanelItem.

Возвращаемое значение

Нет.
