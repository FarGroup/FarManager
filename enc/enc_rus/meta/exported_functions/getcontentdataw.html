GetContentDataW

главная | экспортируемые функции | TODO: Content API
Функция GetContentDataW вызывается для получения от плагина данных контент-колонок для заданного файла.
intptr_t WINAPI GetContentDataW(
  struct GetContentDataInfo *Info
);

Параметры

Info
Указатель на структуру GetContentDataInfo.

Возвращаемое значение

Если плагин устанавливал данные в Info->Values, то верните 1. В противном случае 0.
Смотрите также:
GetContentFieldsW, FreeContentDataW

ПРИМЕЧАНИЕ

Данное API не является окончательным и может быть изменено или удалено в любой момент.
