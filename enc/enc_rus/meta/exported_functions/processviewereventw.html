ProcessViewerEventW

главная | экспортируемые функции | Viewer API
Функция ProcessViewerEventW сообщает плагину о событиях, происходящих во внутренней программе просмотра Far Manager. 
intptr_t WINAPI ProcessViewerEventW(
  const struct ProcessViewerEventInfo *Info
);

Параметры

Info
Указатель на структуру ProcessViewerEventInfo, Far Manager передает информацию о событии.
В начале функции проверьте, что поле ProcessViewerEventInfo.StructSize >= sizeof(ProcessViewerEventInfo).

Возвращаемое значение

Возвращаемое значение зависит от типа события, смотрите описание событий для уточнения информации.
Для необрабатываемых плагином событий, функция должна возвращать 0.
