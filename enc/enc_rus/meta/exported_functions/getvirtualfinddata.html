GetVirtualFindData

главная | экспортируемые функции
Функция GetVirtualFindData может быть использована для получения списка файлов для показа их на другой панели в дополнение к реальным файлам.
int WINAPI GetVirtualFindData(
  HANDLE hPlugin,
  struct PluginPanelItem **pPanelItem,
  int *pItemsNumber,
  const char *Path
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPlugin или OpenFilePlugin.
pPanelItem
Указывает на переменную, которая получит адрес массива структур PluginPanelItem.
pItemsNumber
Указывает на переменную, которая получит число структур PluginPanelItem.
Path
Путь, для которого запрашивается список файлов (текущий каталог на другой панели). Путь завершен обратным слешем ("\").

Возвращаемое значение

В случае успеха должно возвращаться TRUE. Иначе FALSE.

Замечания

Функция GetVirtualFindData может использоваться для реализации "отложенного копирования" файлов, когда файлы, скопированные с панели плагина на файловую панель, показываются на файловой панели сразу же, а физическое копирование файлов происходит позже, при закрытии плагина или после выполнения специальной команды. Такое "отложенное копирование" может быть полезно, например, в плагине, работающем с Arvid.
