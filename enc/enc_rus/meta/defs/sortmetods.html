Метод сортировки

главная | типы и определения

Метод сортировки может принимать одно из следующих значений (перечисление OPENPANELINFO_SORTMODES):
МетодОписание
SM_DEFAULT Сортировка по умолчанию.
SM_UNSORTED Без сортировки.
SM_NAME По имени.
SM_EXT По расширению.
SM_MTIME По дате последней записи в файл.
SM_CTIME По дате создания файла.
SM_ATIME По дате последнего доступа.
SM_SIZE По размеру.
SM_DESCR По описанию.
SM_OWNER По владельцу.
SM_COMPRESSEDSIZE По сжатому размеру.
SM_NUMLINKS По количеству ссылок файлов.
SM_NUMSTREAMS По количеству потоков.
SM_STREAMSSIZE По размеру данных во всех потоках.
SM_NAMEONLY По имени файла без расширения.
SM_CHTIME По дате изменения файла.
