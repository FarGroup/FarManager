$AKey

главная | Макросы | Макроязык | Макрокоманды
Оператор $AKey предназначен для вставки во входной буфер клавиши, вызвавшей макрос (реально нажатое сочетание, которым вызывался макрос). 
$AKey

Пример

Макрос для переключения между экранами. В разделе Common вводятся 10 однотипных макросов AltЦифра: 

$If ( Dialog || MainMenu || Menu || UserMenu || Search )
  $AKey
$Else
  %s = AKey ( 1 );
  %c = substr ( %s, len ( %s ) - 1, 1 );
  F12
  $If ( CheckHotkey ( %c ) )
    eval ( %c )
  $Else
    Esc
    $AKey
  $End
$End
