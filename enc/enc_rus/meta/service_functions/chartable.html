CharTable

главная | сервисные функции
Функция CharTable позволяет получить информацию об установленных таблицах символов. 
int WINAPI CharTable(
  int Command,
  char *Buffer,
  int BufferSize
);

Параметры

Command
Или номер требуемой символьной таблицы, или одна из следующих команд (перечисление FARCHARTABLE_COMMAND):
КомандаОписание
FCT_DETECT Автоматически определить таблицу символов для данного текста
Buffer
Для Command = FCT_DETECT - адрес буфера с текстовыми данными. Иначе - адрес структуры CharTableSet, которая получает информацию о запрошенной таблице символов.
BufferSize
Если Command равна FCT_DETECT,то это должен быть размер буфера с анализируемыми текстовыми данными. Иначе - размер структуры CharTableSet.

Возвращаемое значение

-1, если запрашиваемая таблица отсутствует или не удалось автоопределение. В случае успеха возвращает номер требуемой таблицы и заполняет структуру, указанную Buffer. В режиме FCT_DETECT возвращает номер определенной таблицы и не изменяет данные Buffer.

Примечание

  1. Чтобы перебрать все символьные таблицы, начните с Command равной 0 и увеличивайте ее, пока возвращаемое значение не равно -1.
  2. При неудачном считывании настроек для определенной символьной таблицы структура CharTableSet заполняется данными для OEM (для случая, когда Command не равно FCT_DETECT).
