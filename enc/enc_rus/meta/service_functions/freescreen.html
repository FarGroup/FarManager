FreeScreen

главная | сервисные функции | Miscellaneous API
Функция FreeScreen освобождает память, зарезервированную функцией SaveScreen, без восстановления экрана. 
void WINAPI FreeScreen(
  HANDLE hScreen
);

Параметры

hScreen
Описатель, полученный после вызова SaveScreen. После вызова FreeScreen он больше не может использоваться.

Возвращаемое значение

Нет.
