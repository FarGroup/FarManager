SaveScreen

главная | сервисные функции
Функция SaveScreen сохраняет область экрана. Для восстановления сохраненной области используйте функцию RestoreScreen. 
HANDLE WINAPI SaveScreen(
  int X1,
  int Y1,
  int X2,
  int Y2
);

Параметры

X1,Y1,X2,Y2
Координаты области экрана. Если X2 или Y2 равны -1, то они заменяются соответственно правой или нижней координатой экрана. Таким образом, SaveScreen(0,0,-1,-1) сохранит весь экран.

Возвращаемое значение

Возвращаемое значение используется функцией RestoreScreen. Все указатели, возвращенные SaveScreen , должны быть переданы RestoreScreen, чтобы предотвратить утечку памяти.
