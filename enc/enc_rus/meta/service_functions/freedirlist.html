FreeDirList

главная | сервисные функции | Panel API
Функция FreeDirList освобождает память, выделенную для списка файлов функциями GetDirList. 
void WINAPI FreeDirList(
  struct PluginPanelItem *PanelItem,
  size_t nItemsNumber
);

Параметры

PanelItem
Адрес массива структур PluginPanelItem.
nItemsNumber
Количество элементов в массиве.

Возвращаемое значение

Нет.
