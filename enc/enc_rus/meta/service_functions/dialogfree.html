DialogFree

главная | сервисные функции | Dialog API
Функция DialogFree освобождает память занятую при создании диалога функцией DialogInit. 
void WINAPI DialogFree(
  HANDLE hDlg
);

Параметры

hDlg
Описатель диалога созданный функцией DialogInit.

Возвращаемое значение

Функция не возвращает значение.
