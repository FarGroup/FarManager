FreePluginDirList

главная | сервисные функции | Panel API
Функция FreePluginDirList освобождает память, выделенную для списка файлов эмулируемой файловой системы функцией GetPluginDirList. 
void WINAPI FreePluginDirList(
  HANDLE hPanel,
  struct PluginPanelItem *PanelItem,
  size_t nItemsNumber
);

Параметры

hPanel
Текущий описатель панели.
PanelItem
Адрес массива структур PluginPanelItem.
nItemsNumber
Количество элементов в массиве.

Возвращаемое значение

Нет.
