CRegExp.SetExpr

главная | Custom API | регэкспы
Устанавливает и компилирует регулярное выражение.

Метод класса:

bool SetExpr(char *Expr);

DLL интерфейс:

BOOL WINAPI reSetExpr(PRegExp re, char *expr)

Элементы

re
Указатель на объект
expr
Строка с регулярным выражением, записанным по правилам колорера. Если вы ищете в кодировке иной, чем OEM (dos866), вы должны во-первых передавать в этот метод преобразованную в вашу кодировку строку, а во-вторых вызвать функцию SetCodePage с адресом таблицы перекодировки из вашей кодировки в OEM.

Возвращаемое значение

Возвращает TRUE или FALSE в зависимости от успеха операции.

Замечания

Возможно использование расширенного метода geterror для более подробного обнаружения ошибки...
