CRegExp.SetCodePage

главная | Custom API | регэкспы
[ отсутствует в colorer FreeCraze ]
Устанавливает кодовую страницу для работы с отличными от OEM кодировками.

Метод класса:

void SetCodePage(char *Table);

DLL интерфейс:

void WINAPI reSetCodePage(PRegExp re, char *Table);

Элементы

re
Указатель на объект
Table
Таблица для перекодировки из целевой в кодировку OEM

Возвращаемое значение

нет

Замечания

Для полноценной работы регулярных выражений в иной кодировке необходимо устанавливать само регулярное выражение также в целевой кодировке. Заметьте, что вы можете перекомпилировать класс так, чтобы по умолчанию он использовал отличную от OEM кодировку. В этом случае метод setcodepage также будет работать, но перекодировочные таблицы должны быть относительно кодировки, в которой скомпилирован класс CRegExp.
