CRegExp.SetNoMoves

главная | Custom API | регэкспы
Устанавливает возможность перемещения по целевой строке.

Метод класса:

bool SetNoMoves(bool Moves);

DLL интерфейс:

BOOL WINAPI reSetNoMoves(PRegExp re, BOOL Moves);

Элементы

re
Указатель на объект
Moves
Если TRUE, то функция сканирования не перемещается по целевой строке. Иначе производится попытка найти совпадение на каждой позиции в строке

Возвращаемое значение

Возвращает успех/неуспех установки параметра.

Замечания

При разрешении перемещений по целевой строке фактически производится вызов функции сканирования для каждой позиции в строке с первой по последнюю и на любой из них ищется совпадение. Если вам не нужна эта возможность, не забудьте установить SetNoMoves(TRUE);
