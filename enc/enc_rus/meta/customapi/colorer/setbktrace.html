CRegExp.SetBkTrace

главная | Custom API | регэкспы
Устанавливает отношения для оператора \yN

Метод класса:

bool SetBkTrace(char *Str, PMatches Trace);

DLL интерфейс:

BOOL WINAPI reSetBkTrace(PRegExp re, char *str, PMatches trace);

Элементы

re
Указатель на объект
Str
Строка для установки отношения
trace
набор совпадений скобок для предыдущего регулярного выражения

Возвращаемое значение

Возвращает TRUE или FALSE в зависимости от успеха операции.

Замечания

Эта функция используется для установки связи между разными регулярными выражениями (используется к примеру плагином колорер) - через оператор \yN где N - номер скобки в первом регулярном выражении для ссылки.
