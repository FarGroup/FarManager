CRegExp.isok

главная | Custom API | регэкспы
Возвращает результат последней компиляции.

Метод класса:

bool isok();

DLL интерфейс:

BOOL WINAPI reIsok(PRegExp re);

Элементы

re
Указатель на объект

Возвращаемое значение

Возвращает TRUE или FALSE в зависимости от успеха последней компиляции.

Замечания

Возможно использование расширенного метода geterror для более подробного обнаружения ошибки...
