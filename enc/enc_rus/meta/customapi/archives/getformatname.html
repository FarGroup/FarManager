GetFormatName

главная | Custom API | поддержка архивов
Получить имя формата архива. 
BOOL WINAPI GetFormatName(
  int Type,
  char *FormatName,
  char *DefaultExt
);

Параметры

Type
тип архива
FormatName
имя формата; оно будет использовано при сохранении параметров в реестре и при выборе желательного формата архива.
DefaultExt
расширение файла по умолчанию для данного формата (без точки); используется для ускорения распознавания формата.

Возвращаемое значение

В случае успеха возвращается TRUE. Если указанное значение типа архива не поддерживается, то должно возвращаться FALSE.

Пример

Пример получения имени формата ZIP-файла (ZIP.CPP)

BOOL WINAPI _export GetFormatName(int Type,char *FormatName,char *DefaultExt)
{
  if (Type==0)
  {
    strcpy(FormatName,"ZIP");
    strcpy(DefaultExt,"zip");
    return(TRUE);
  }
  return(FALSE);
}
