GetDefaultCommands

главная | Custom API | поддержка архивов
Получить строки команд архиватора, используемые по умолчанию 
BOOL WINAPI GetDefaultCommands(
  int Type,
  int Command,
  char *Dest
);

Параметры

Type
Тип архива
Command
Номер команды архиватора:
НомерОписание
0extract (распаковка)
1extract without path (распаковка без путей)
2test (тестирование)
3delete (удаление)
4comment (архивный комментарий)
5comment files (файловый комментарий)
6convert to SFX (преобразование в SFX)
7lock (запрет модификации)
8add recovery record (защита от повреждений)
9recover (восстановление архива)
10add files (добавление файлов)
11move files (перенос файлов)
12add files and folders (добавление файлов и папок)
13move files and folders (перенос файлов и папок)
14mask to select all files (маска "Все файлы")

В настройках плагина MultiArc это выглядит так:

Dest
Буфер для копирования команд. MultiArc ожидает строку не более 511 байт.

Возвращаемое значение

В случае успеха возвращается TRUE. Если указанный тип архива не поддерживается, то FALSE. Если тип поддерживается, но требуемая команда отсутствует, то верните TRUE и установите Dest в пустую строку.
