LoadFormatModule

главная | Custom API | поддержка архивов
Вызывается, когда модуль плагина второго уровня уже загружен. Эта функция дополнительная и может быть опущена. 
DWORD WINAPI LoadFormatModule(
  const char *ModuleName
);

Параметры

ModuleName
имя модуля плагина второго уровня (полный путь к FMT-файлу)

Возвращаемое значение

Должно быть 0. В будущем может использоваться, чтобы возвратить информацию о плагине второго уровня.

Примечание

Эта функция вызывается один раз, когда MultiArc загружает плагин второго уровня (см. исходные тексты MultiArc - файл ARCPLG.CPP, функция ArcPlugins::LoadFmtModules)
