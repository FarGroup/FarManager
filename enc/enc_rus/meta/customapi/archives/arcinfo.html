ArcInfo

главная | Custom API | поддержка архивов
Эта структура передается CloseArchive. 
struct ArcInfo
{
  int SFXSize;
  int Volume;
  int Comment;
  int Recovery;
  int Lock;
  DWORD Flags;
  DWORD Reserved;
  int Chapters;
};

Элементы

SFXSize
размер модуля SFX
Volume
Флаг тома
Comment
Присутствует архивный комментарий
Recovery
Присутствует защита от повреждений
Lock
Архив защищен от изменений
Flags
Дополнительные архивные информационные флаги. Поле Flags может быть комбинацией следующих значений:
СтруктураОписание
AF_AVPRESENT Присутствует электронная подпись (информация о подлинности архива)
AF_IGNOREERRORS Код завершения архивных команд должен игнорироваться для этого архива.
AF_HDRENCRYPTED Заголовок архива шифрованный.
Reserved
Зарезервировано.
Chapters
Количество глав (chapter) в архиве.

Замечания

Структура ArcInfo передается в CloseArchive инициализированная нулями.
