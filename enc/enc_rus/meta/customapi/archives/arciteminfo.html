ArcItemInfo

главная | Custom API | поддержка архивов
Эта структура передается в функцию GetArcItem (поля уже заполнены 0). 
struct ArcItemInfo
{
  char HostOS[32];
  char Description[256];
  int Solid;
  int Comment;
  int Encrypted;
  int DictSize;
  int UnpVer;
  int Chapter;
};

Элементы

HostOS
Операционная система для распаковки или пустая строка, если она неизвестна
Description
Описание элемента или пустая строка
Solid
Флаг, указывающий на непрерывный архив.
Comment
Установите, если присутствует файловый комментарий
Encrypted
Установите, если файл зашифрован
DictSize
Размер словаря или 0, если он неизвестен
UnpVer
Версия, необходимая для распаковки (HighNumber*256+LowNumber) или 0 если неизвестно
Chapter
глава (chapter) в архиве, которой принадлежит файл.
