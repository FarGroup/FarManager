Здесь описываются некоторые моменты оформления и организации статей для Энциклопедии.
  1. Ссылка "внутрь" Энциклопедии (например, для структуры PluginStartupInfo):

    <a href="mk:@MSITStore:FarEncyclopedia.ru.chm::/html/structures/pluginstartupinfo.html">PluginStartupInfo</a>

    Здесь следует учесть структуры каталогов FarEncyclopedia.ru.chm и принципы наименования статей:

    • имена html-файлов в основном соответствуют названиям функций и структур;
    • все html-файлы внутри FarEncyclopedia.ru.chm в нижнем регистре;
    • html-файлы раскиданы по подкаталогам в соответствии со своими функциональными назначениями, например, основные структуры располагаются в FarEncyclopedia.ru.chm::/html/structures/, структуры Dialog API в FarEncyclopedia.ru.chm::/html/dialogapi/structures/ и т.п.


  2. Ссылка на WinAPI функцию (не описанную в Энциклопедии), для Вашей статьи, например для функции GetLastError:

    <a class="msdocs">GetLastError</a>

    при помощи javascript такая ссылка автоматически преобразуется в полную ссылку для поиска на сайте Microsoft: GetLastError.
