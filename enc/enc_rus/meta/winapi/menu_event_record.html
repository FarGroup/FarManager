MENU_EVENT_RECORD

главная | структуры | win32 структуры | INPUT_RECORD
Структура MENU_EVENT_RECORD содержит сообщения меню в консольной структуре INPUT_RECORD. Эти сообщения используются системой и должны игнорироваться. 
typedef struct _MENU_EVENT_RECORD {
    UINT dwCommandId;
} MENU_EVENT_RECORD, *PMENU_EVENT_RECORD;

Элементы

dwCommandId
Зарезервировано.
