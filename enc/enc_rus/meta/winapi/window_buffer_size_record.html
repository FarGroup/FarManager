WINDOW_BUFFER_SIZE_RECORD

главная | структуры | win32 структуры
Структура WINDOW_BUFFER_SIZE_RECORD используется консольной структурой INPUT_RECORD для информирования о смене в размере экранного буфера консоли. 
typedef struct _WINDOW_BUFFER_SIZE_RECORD {
    COORD dwSize;
} WINDOW_BUFFER_SIZE_RECORD;

Элементы

dwSize
Указывает новый размер экранного буфера в символах (структура COORD).

Замечания

Эти сообщения добавляются во входной буфер, когда консоль находится в режиме window-aware (ENABLE_WINDOW_INPUT).
