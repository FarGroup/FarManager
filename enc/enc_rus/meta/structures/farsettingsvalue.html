FarSettingsValue

главная | структуры | Settings API
Структура FarSettingsValue используется в функции SettingsControl для управления настройками. 
struct FarSettingsValue
{
  size_t StructSize;
  size_t Root;
  const wchar_t *Value;
};

Элементы

StructSize
Это поле должно содержать размер структуры FarSettingsValue. Перед использованием структуры плагин должен заполнить поле.
Root
Описатель ключа, откуда брать информацию.
Value
Имя.
