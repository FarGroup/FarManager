FarMacroLoad

главная | структуры | Macro API
Структура FarMacroLoad используется в функции MacroControl для загрузки макрофайлов в Far Manager. 
struct FarMacroLoad
{
  size_t StructSize;
  const wchar_t *Path;
  unsigned __int64 Flags;
};

Элементы

StructSize
Это поле должно содержать размер структуры FarMacroLoad. Перед использованием структуры плагин должен заполнить поле.
Path
Перечень путей, с которых должны грузиться макрофайлы.
  • Разделитель путей - точка с запятой.
  • В путях можно использовать переменные окружения.
Flags
На данный момент не используется. Установите в 0.
