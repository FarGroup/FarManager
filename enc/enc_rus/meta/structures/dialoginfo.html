DialogInfo

главная | структуры | Dialog API
Структура DialogInfo позволяет узнать информацию о диалоге, является параметром в DM_GETDIALOGINFO. 
struct DialogInfo
{
  size_t StructSize;
  GUID Id;
  GUID Owner;
};

Элементы

StructSize
Это поле должно содержать размер структуры DialogInfo. Перед использованием структуры плагин должен заполнить поле.
Id
Уникальный идентификатор диалога.
Owner
Поле содержит GUID плагина, создавшего диалог или GUID Far Manager, для созданного им диалога.
