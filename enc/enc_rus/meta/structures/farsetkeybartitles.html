FarSetKeyBarTitles

главная | структуры | Panel API
Структура FarSetKeyBarTitles позволяет переопределять заголовки функциональных клавиш. Передаётся в Far Manager, например, функцией ECTL_SETKEYBAR. 
struct FarSetKeyBarTitles
{
  size_t StructSize;
  struct KeyBarTitles *Titles;
};

Элементы

StructSize
Это поле должно содержать размер структуры FarSetKeyBarTitles. Перед использованием структуры плагин должен заполнить поле.
Titles
Содержит массив структур типа KeyBarTitles, с новыми заголовками функциональных клавиш.
