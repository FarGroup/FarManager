FarSettingsCreate

главная | структуры | Settings API
Структура FarSettingsCreate используется в функции SettingsControl для управления настройками. 
struct FarSettingsCreate
{
  size_t StructSize;
  GUID Guid;
  HANDLE Handle;
};

Элементы

StructSize
Это поле должно содержать размер структуры FarSettingsCreate. Перед использованием структуры плагин должен заполнить поле.
Guid
Поле должно содержать GUID плагина, для которого создаётся объект "настройки".
Handle
Far Manager в это поле поместит описатель созданных настроек для указанного в Guid плагина.
