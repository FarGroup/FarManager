FarSetColors

главная | структуры
Структура FarSetColors используется в функции AdvControl для изменения цветовой схемы FAR Manager. 
struct FarSetColors {
  DWORD Flags;
  int StartIndex;
  int ColorItem;
  LPBYTE Colors;
};

Элементы

Flags
Комбинация флагов (перечисление FARCOLORFLAGS):
ФлагОписание
FCLR_REDRAW После изменений цветовой схемы обновить экран.
StartIndex
Начальная позиция индекса цветовой схемы для замены.
ColorItem
Количество позиций цветовой для замены.
Colors
Указатель на массив цветовой схемы - байтовый массив цветовых атрибутов.
