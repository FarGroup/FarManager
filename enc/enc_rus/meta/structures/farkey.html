FarKey

главная | структуры | Basic API
Структура FarKey описывает код клавиши, используется, например, в функции Menu. 
struct FarKey
{
  WORD VirtualKeyCode;
  DWORD ControlKeyState;
};

Элементы

VirtualKeyCode
Поле аналогично значению KEY_EVENT_RECORD.wVirtualKeyCode.
ControlKeyState
Поле аналогично значению KEY_EVENT_RECORD.dwControlKeyState.
