ViewerMode

главная | структуры
Структура ViewerMode используется для получения дополнительной информации о текущем состоянии программы просмотра FAR. 
struct ViewerMode
{
   int UseDecodeTable;
   int TableNum;
   int AnsiMode;
   int Unicode;
   int Wrap;
   int TypeWrap;
   int Hex;
   DWORD Reserved[4];
};

Элементы

UseDecodeTable
Если TRUE, то используется таблица перекодировки.
TableNum
Номер пользовательской таблицы перекодировки. Имеет смысл, если UseDecodeTable равно TRUE.
AnsiMode
Если TRUE, то текущая кодировка в программе просмотра - ANSI.
Unicode
Если TRUE, то текущая кодировка в программе просмотра - Unicode.
Wrap
Включена (TRUE)/выключена (FALSE) свертка.
TypeWrap
Тип свертки.
Hex
Если TRUE, то программа просмотра в hex-режиме.
Reserved
Зарезервировано для будущего использования.
