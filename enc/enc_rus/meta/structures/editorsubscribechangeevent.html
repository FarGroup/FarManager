EditorSubscribeChangeEvent

главная | структуры | Editor API
Структура EditorSubscribeChangeEvent используется в функции EditorControl для подписки (или отписки) на EE_CHANGE события редактора. 
struct EditorSubscribeChangeEvent
{
  size_t StructSize;
  GUID PluginId;
};

Элементы

StructSize
Это поле должно содержать размер структуры EditorSubscribeChangeEvent. Перед использованием структуры плагин должен заполнить поле.
PluginId
GUID плагина подписчика.
