FarList

главная | структуры | Dialog API
Структура FarList описывает список для элементов диалога DI_LISTBOX и DI_COMBOBOX. 
struct FarList
{
  size_t StructSize;
  size_t ItemsNumber;
  struct FarListItem *Items;
};

Элементы

StructSize
Это поле должно содержать размер структуры FarList. Перед использованием структуры плагин должен заполнить поле.
ItemsNumber
Количество элементов списка Items.
Items
Указатель на массив элементов списка FarListItem.
