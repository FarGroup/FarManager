PluginStartupInfo

главная | структуры
Структура PluginStartupInfo используется в функции SetStartupInfo для передачи плагину различной важной информации. 
struct PluginStartupInfo
{
  int StructSize;

  char ModuleName[NM];
  int ModuleNumber;
  const char *RootKey;
  FARAPIMENU             Menu;
  FARAPIDIALOG           Dialog;
  FARAPIMESSAGE          Message;
  FARAPIGETMSG           GetMsg;
  FARAPICONTROL          Control;
  FARAPISAVESCREEN       SaveScreen;
  FARAPIRESTORESCREEN    RestoreScreen;
  FARAPIGETDIRLIST       GetDirList;
  FARAPIGETPLUGINDIRLIST GetPluginDirList;
  FARAPIFREEDIRLIST      FreeDirList;

  // FAR >= 1.50
  FARAPIVIEWER           Viewer;
  FARAPIEDITOR           Editor;
  FARAPICMPNAME          CmpName;

  // FAR >= 1.52
  FARAPICHARTABLE        CharTable;
  FARAPITEXT             Text;

  // FAR >= 1.60
  FARAPIEDITORCONTROL    EditorControl;

  // FAR >= 1.70
  FARSTANDARDFUNCTIONS  *FSF;

  FARAPISHOWHELP         ShowHelp;
  FARAPIADVCONTROL       AdvControl;
  FARAPIINPUTBOX         InputBox;
  FARAPIDIALOGEX         DialogEx;
  FARAPISENDDLGMESSAGE   SendDlgMessage;
  FARAPIDEFDLGPROC       DefDlgProc;
  DWORD                  Reserved1;
  DWORD                  Reserved2;
};

Элементы

StructSize
Размер структуры. Если в будущем будут добавлены новые поля, это поле позволит определить это.
ModuleName
Полное имя плагина с путем.
ModuleNumber
Номер плагина. Передается как параметр в некоторых функциях.
RootKey
Ключ Реестра, где плагин может сохранять свои параметры. Одинаков и для HKEY_CURRENT_USER и для HKEY_LOCAL_MACHINE. Обычно это "Software\Far\Plugins", но вы не должны указывать эту строку непосредственно, т.к. она может быть изменена или может стать опциональной в будущем. Например, если FAR запущен с ключем '/u user1', то RootKey соответственно будет содержать строку "Software\Far\Users\user1\Plugins".

Не сохраняйте параметры прямо в RootKey, создайте здесь свой собственный подключ.

Замечания

Версии FAR Manager:
  • ниже 1.50 (с поддержкой плагинов) - не содержат поля после FreeDirList;
  • ниже 1.52 - не содержат поля после CmpName;
  • ниже 1.60 - не содержат поля после Text;
  • ниже 1.65 - не содержат поля после EditorControl
Таким образом, если вы собираетесь использовать функции Viewer, Editor или любую другую после нее, вы должны сначала проверить поле StructSize, чтобы гарантировать, что эти функции обеспечиваются текущей версией FAR'а.
