SetFindListInfo

главная | структуры | Panel API
Структура SetFindListInfo передает информацию о найденных объектах в функцию SetFindListW. 
struct SetFindListInfo
{
  size_t StructSize;
  HANDLE hPanel;
  const struct PluginPanelItem *PanelItem;
  size_t ItemsNumber;
};

Элементы

StructSize
Это поле содержит размер структуры SetFindListInfo.
hPanel
Описатель панели, возвращаемый функцией OpenW.
PanelItem
Указатель на массив структур PluginPanelItem. Каждая структура описывает файл для помещения в панель.
ItemsNumber
Число элементов в массиве PanelItem.
