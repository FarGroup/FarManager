PanelRedrawInfo

главная | структуры
Структура PanelRedrawInfo может быть использована для определения новой позиции при перерисовке панели с помощи функции Control. 
struct PanelRedrawInfo
{
  int CurrentItem;
  int TopPanelItem;
};

Элементы

CurrentItem
Текущая позиция на панели (начиная с 0).
TopPanelItem
Позиция верхнего пункта панели (начиная с 0).
