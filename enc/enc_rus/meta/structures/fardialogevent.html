FarDialogEvent

главная | структуры | Dialog API
Структура FarDialogEvent содержит информацию о событии, поступающем в функцию ProcessDialogEventW. 
struct FarDialogEvent
{
  size_t StructSize;
  HANDLE hDlg;
  intptr_t Msg;
  intptr_t Param1;
  void *Param2;
  intptr_t Result;
};

Элементы

StructSize
Это поле содержит размер структуры FarDialogEvent.
hDlg
Указатель на диалог.
Msg
Одно из сообщений или событий.
Param1
Параметр 1.
Param2
Параметр 2.
Result
Значение, возвращаемое обработчиком диалога.
