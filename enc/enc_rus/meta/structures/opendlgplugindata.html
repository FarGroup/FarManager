OpenDlgPluginData

главная | структуры | Dialog API
Указатель на структуру OpenDlgPluginData передаётся в функцию OpenW плагина при вызове его из диалога. 
struct OpenDlgPluginData
{
  size_t StructSize;
  HANDLE hDlg;
};

Элементы

StructSize
Это поле содержит размер структуры OpenDlgPluginData.
hDlg
Описатель диалога, из которого вызван плагин.
