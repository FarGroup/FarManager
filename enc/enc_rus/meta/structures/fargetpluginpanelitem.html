FarGetPluginPanelItem

главная | структуры | Panel API
Структура FarGetPluginPanelItem используется для запроса информации о файловом объекте в функции PanelControl. 
struct FarGetPluginPanelItem
{
  size_t StructSize;
  size_t Size;
  struct PluginPanelItem *Item;
};

Элементы

StructSize
Это поле должно содержать размер структуры FarGetPluginPanelItem. Перед использованием структуры плагин должен заполнить это поле.
Size
Размер буфера, на который указывает поле Item, в байтах.
Item
Буфер, выделенный плагином для размещения запрашиваемого элемента панели PluginPanelItem. Вслед за собственно структурой в буфер будут помещены значения переменной длины, на которые указывают поля структуры.

Замечания

Выделенный буфер должен быть освобожден плагином. Смотрите пример в функции PanelControl.
