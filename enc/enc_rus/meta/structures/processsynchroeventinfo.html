ProcessSynchroEventInfo

главная | структуры | Miscellaneous API
Структура ProcessSynchroEventInfo содержит информацию о событии, поступающем в функцию ProcessSynchroEventW. 
struct ProcessSynchroEventInfo
{
  size_t StructSize;
  intptr_t Event;
  void *Param;
};

Элементы

StructSize
Это поле содержит размер структуры ProcessSynchroEventInfo.
Event
Тип события. Одно из следующих значений (перечисление SYNCHRO_EVENTS):
СобытиеОписание
SE_COMMONSYNCHRO Событие приходит, если плагином вызывается функция Info.AdvControl(...,ACTL_SYNCHRO,Param1,Param2).
Param указывает на переданные данные в параметре Info.AdvControl(...,Param2).
Функция ProcessSynchroEventW должна вернуть 0.
Param
Значение указателя зависит от типа события. Смотрите описание событий для уточнения информации.
