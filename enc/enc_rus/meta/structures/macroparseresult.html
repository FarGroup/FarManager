MacroParseResult

главная | структуры | Macro API
Структура MacroParseResult используется в функции MacroControl для возврата плагину результатов проверки макропоследовательности. 
struct MacroParseResult
{
  size_t StructSize;
  DWORD ErrCode;
  COORD ErrPos;
  const wchar_t *ErrSrc;
};

Элементы

StructSize
Это поле должно содержать размер структуры MacroParseResult. Перед использованием структуры плагин должен заполнить поле.
ErrCode
Результат проверки (перечисление FARMACROPARSEERRORCODE):
ФлагОписание
MPEC_SUCCESS Ошибок нет.
MPEC_ERROR Имеются ошибки.
ErrPos
Позиция ошибки в исходном тексте.
ErrSrc
Текстовое описание ошибки.
