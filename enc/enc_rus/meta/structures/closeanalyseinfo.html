CloseAnalyseInfo

главная | структуры | Panel API
Структура CloseAnalyseInfo используется Far Manager'ом в функции CloseAnalyseW для освобождения ресурсов. 
struct CloseAnalyseInfo
{
  size_t StructSize;
  HANDLE Handle;
};

Элементы

StructSize
Это поле содержит размер структуры CloseAnalyseInfo.
Handle
Описатель, созданный в функции AnalyseW.
