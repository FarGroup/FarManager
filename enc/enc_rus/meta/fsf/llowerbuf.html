LLowerBuf

главная | стандартные функции
Функция FSF.LLowerBuf преобразует массив символов указанной длины, (который может содержать нулевые символы) в нижний регистр. 
void WINAPI LLowerBuf(
  wchar_t *Buf,
  int Length
);

Параметры

Buf
Указатель на обрабатываемый массив.
Length
Размер массива.

Возвращаемое значение

Нет.
