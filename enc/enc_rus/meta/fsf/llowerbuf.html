LLowerBuf

главная | FarStandardFunctions
Функция FSF.LLowerBuf преобразует массив символов указанной длины, (который может содержать нулевые символы) в нижний регистр. Функция работает с кодировкой OEM. 
void WINAPI LLowerBuf(
  char *Buf,
  int Length
);

Параметры

Buf
Указатель на обрабатываемый массив.
Length
Размер массива в байтах.

Возвращаемое значение

Нет.
