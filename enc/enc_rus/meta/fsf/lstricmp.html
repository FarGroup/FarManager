LStricmp

главная | FarStandardFunctions
Функция FSF.LStricmp сравнивает две строки без учета регистра. Функция работает с кодировкой OEM. 
int WINAPI LStricmp(
  const char *Str1,
  const char *Str2
);

Параметры

Str1, Str2
Указатели на строки, которые вы хотите сравнить.

Возвращаемое значение

Функция возвращает:
-1 - если s1 < s2
1 - если s1 > s2
0 - если s1 == s2
