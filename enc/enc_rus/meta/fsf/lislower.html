LIsLower

главная | FarStandardFunctions
Функция FSF.LIsLower проверяет, является ли символ с указанным кодом буквой в нижнем регистре. Функция работает с кодировкой OEM. 
int WINAPI LIsLower(
  unsigned Ch
);

Параметры

Ch
Код символа для проверки.

Возвращаемое значение

Если символ с указанным кодом является буквой в нижнем регистре, то вернется TRUE, иначе - FALSE.
