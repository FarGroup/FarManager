MkLink

главная | FarStandardFunctions
Функция FSF.MkLink создает жесткие и символические связи, а также монтирует локальные диски на файловую систему. Функция работает только под WinNT4 и выше. 
int WINAPI MkLink(
  const char *Src,
  const char *Dest,
  DWORD Flags
);

Параметры

Src
Имя файлового объекта, связь с которым создается.
Dest
Имя создаваемой связи.
Flags
Тип операции. Одно из (перечисление MKLINKOP):
ОперацияОписание
FLINK_HARDLINK Создать жесткую связь (hardlink).
FLINK_SYMLINK Создать символическую связь (symlink).
FLINK_VOLMOUNT монтировать локальный диск на файловую систему.

В дополнении к вышеуказанным операциям можно указать флаг:

ФлагОписание
FLINK_SHOWERRMSG Выводить сообщения об ошибках.
FLINK_DONOTUPDATEPANEL После создания связи не обновлять панели.

Возвращаемое значение

1 - связь создана.

0 - ошибка при создании связи:

  • Для жестких связей:
    • Src или Dest находятся на разных разделах;
    • раздел не NTFS;
    • раздел не локальный;
    • Src не существует или не является файлом.
    • Dest уже существует;
  • Для символических связей:
    • Src или Dest находятся не на локальных разделах;
    • раздел не NTFS 5.0;
    • Src не существует или не является каталогом;
    • Dest существует, но не пустой каталог.
  • Для монтирования дисков:
    • Src или Dest находятся не на локальных разделах;
    • раздел для Dest не NTFS 5.0;
    • Src не существует или диск является не локальным;
    • Dest существует, но не пустой каталог.

Примечание

  1. Связи создаются по следующим правилам:
    • жесткие связи (hardlink) создаются только для файлов в пределах текущего NTFS-раздела (NT4/Win2K/NTFS);
    • символические связи (symlink) создаются только для каталогов в пределах локальных NTFS-разделов (Win2K/NTFS 5.0);
    • монтирование локальных дисков на файловую систему только на NTFS-раздел (Win2K/NTFS 5.0)
  2. Если в качестве Src было передано значение, например "C:", то вместо символической связи будет монтирован диск.
  3. Если для монтирования диска Dest оканчивается слешем, то в Dest будет создан каталог "disk_N" (где N - буква диска).
  4. Напрямую делать символические связи на папки CD-ROM нельзя, но можно это ограничение обойти, смонтировав CD-ROM диск на папку NTFS-раздела (т.н. Mount Points). И уже из этой папки сделать нужную связь.
