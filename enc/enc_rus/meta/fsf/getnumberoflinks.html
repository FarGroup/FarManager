GetNumberOfLinks

главная | FarStandardFunctions
Функция FSF.GetNumberOfLinks возвращает количество жестких связей (hard links) для заданного файла. 
int WINAPI GetNumberOfLinks(
  const char *Name
);

Параметры

Name
Имя требуемого файла.

Возвращаемое значение

В случае успеха функция возвращает количество жестких связей или 0. Для файловых систем, отличных от NTFS это значение всегда равно 1.
