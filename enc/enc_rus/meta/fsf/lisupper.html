LIsUpper

главная | стандартные функции
Функция FSF.LIsUpper проверяет, является ли символ с указанным кодом буквой в верхнем регистре. 
int WINAPI LIsUpper(
  wchar_t Ch
);

Параметры

Ch
Код символа для проверки.

Возвращаемое значение

Если символ с указанным кодом является буквой в верхнем регистре, то вернётся TRUE, иначе - FALSE.
