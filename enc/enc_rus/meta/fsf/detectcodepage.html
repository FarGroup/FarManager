DetectCodePage

главная | стандартные функции
Функция FSF.DetectCodePage возвращает кодовую страницу указанного файла в соответствии с результатом детектирования, произведённого Far Manager. 
uintptr_t WINAPI DetectCodePage(
  struct DetectCodePageInfo *Info
);

Параметры

Info
Указатель на структуру DetectCodePageInfo

Возвращаемое значение

Кодовая страница, определённая для указанного файла.
