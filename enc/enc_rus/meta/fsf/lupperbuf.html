LUpperBuf

главная | FarStandardFunctions
Функция FSF.LUpperBuf преобразует массив символов указанной длины, (который может содержать нулевые символы) в верхнем регистр. Функция работает с кодировкой OEM. 
void WINAPI LUpperBuf(
  char *Buf,
  int Length
);

Параметры

Buf
Указатель на обрабатываемый массив.
Length
Размер массива в байтах.

Возвращаемое значение

Нет.
