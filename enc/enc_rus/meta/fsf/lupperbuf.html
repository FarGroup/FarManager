LUpperBuf

главная | стандартные функции
Функция FSF.LUpperBuf преобразует массив символов указанной длины, (который может содержать нулевые символы) в верхнем регистр. 
void WINAPI LUpperBuf(
  wchar_t *Buf,
  int Length
);

Параметры

Buf
Указатель на обрабатываемый массив.
Length
Размер массива.

Возвращаемое значение

Нет.
