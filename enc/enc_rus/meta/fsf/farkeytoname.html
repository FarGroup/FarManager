FarKeyToName

главная | FarStandardFunctions
Функция FSF.FarKeyToName конвертирует код клавиши, представленной во внутреннем формате FAR Manager, в строку. 
BOOL WINAPI FarKeyToName(
  int Key,
  char *KeyText,
  int Size
);

Параметры

Key
Код клавиши во внутреннем формате FAR.
KeyText
Указатель на строку, куда будет помещен результат преобразования.
Size
Максимальное количество символов, которое можно поместить в KeyText, без заключительного нуля, или 0 для копирования всей получаемой строки.

Возвращаемое значение

В случае успеха возвращается TRUE. Если код клавиши (Key) не известен FAR Manager, то возвращается FALSE.

Примечание

Если вы передали 0 в качестве Size, то размер KeyText должен быть не меньше 32 байтов.
