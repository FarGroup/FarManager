InitMenuItems

главная | дополнения
Функция InitMenuItems предназначена для преобразования массива структур InitMenuItem в FarMenuItem.

Внимание!

  • Функция InitMenuItems не входит в стандартное API.
void InitMenuItems(
         const struct InitMenuItem *Init,
         struct FarMenuItem *Item,
         int ItemsNumber)
{
  int I;
  struct FarMenuItem *PItem=Item;
  const struct InitMenuItem *PInit=Init;
  for (I=0; I < ItemsNumber; I++,PItem++,PInit++)
  {
    PItem->Selected=PInit->Selected;
    PItem->Checked=PInit->Checked;
    PItem->Separator=PInit->Separator;
    if ((unsigned int)PInit->Text < 2000)
      strcpy(PItem->Text,GetMsg((unsigned int)PInit->Text));
    else
      strcpy(PItem->Text,PInit->Text);
  }
}

Параметры

Init
Указатель на массив структур InitMenuItem. Каждая структура описывает пункт меню.
Item
Указатель на массив структур FarMenuItem для инициализации. Каждая структура будет описывать пункт меню.
ItemsNumber
Количество элементов массива Init.

Возвращаемое значение

Нет.

Замечания

Введение функции InitMenuItems обусловлено следующим:
Поскольку переменная FarMenuItem.Text имеет большой размер, прямая инициализация массива FarMenuItem, может заметно увеличить потребление памяти.
