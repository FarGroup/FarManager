InitMenuItem

главная | структуры | дополнения
Структура InitMenuItem описывает один пункт меню. Эта структура не является стандартной - это аналог структуры FarMenuItem. Отличие только в члене Text.

Внимание!

  • Структура InitMenuItem не входит в стандартное API.
struct InitMenuItem
{
  char* Text;
  int Selected;
  int Checked;
  int Separator;
};

Замечания

Введение структуры InitMenuItem, как аналога FarMenuItem обусловлено следующим:
Поскольку переменная FarMenuItem.Text имеет большой размер, прямая инициализация массива FarMenuItem, может заметно увеличить потребление памяти.

Для преобразования InitMenuItem в FarMenuItem можно воспользоваться функцией InitMenuItems.
