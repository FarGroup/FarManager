Собирается всё просто:
структура папок такая-

+enc - просто пустая папка - именно с таким именем
+Enc_SVN - в этой папке будут локальный копии SVN (имя папки пофиг)
|--enc_eng - тут trunk анг енц (имя папки важно)
|--enc_rus - тут trunk рус енц (имя папки важно)
|--tools   - тут trunk tools (имя папки важно)

запускаем
perl tool.make_chm.pl

В папке enc создадутся все html и проект для HCW. Потом уже просто
компилим этот проект с помощью hсw.
