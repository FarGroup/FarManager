OpenMacroInfo

главная | структуры
Структура OpenMacroInfo используется в функции OpenW для передачи значений из макропоследовательности плагину. 
struct OpenMacroInfo
{
  size_t StructSize;
  struct FarMacroValue Value;
};

Элементы

StructSize
Это поле должно содержать размер структуры OpenMacroInfo.
Value
Значение макропеременной, структура FarMacroValue.
