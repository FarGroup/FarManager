FarMacroValue

главная | структуры
Структура FarMacroValue описывает макропеременную. 
struct FarMacroValue
{
  enum FARMACROVARTYPE type;
  union
  {
    __int64  i;
    double   d;
    const wchar_t *s;
  }
#ifndef __cplusplus
  Value
#endif
  ;
};

Элементы

type
Тип переменной, одно из следующих значений (перечисление FARMACROVARTYPE):
ФлагОписание
FMVT_UNKNOWN Неопределнный тип. Value.i = 0.
FMVT_INTEGER Число. Компонента Value.i.
FMVT_STRING Строка. Компонента Value.s.
FMVT_DOUBLE Вещественное значение. Компонента Value.d.
Value
Значение.
