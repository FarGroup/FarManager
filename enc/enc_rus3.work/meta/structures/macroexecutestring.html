MacroExecuteString

главная | структуры | Macro API
Структура MacroExecuteString используется в функции MacroControl для передачи исполняемого скрипта в Far Manager. 
struct MacroExecuteString
{
  size_t StructSize;
  const wchar_t *SequenceText;
  unsigned __int64 Flags;
  size_t Count;
  const struct FarMacroValue *Values;
};

Элементы

StructSize
Это поле должно содержать размер структуры MacroExecuteString. Перед использованием структуры плагин должен заполнить поле.
SequenceText
Строка, содержащая скрипт в текстовом виде. Если строка начинается символом @, то её последующая часть интерпретируется как имя файла-скрипта (допустимо использование переменных окружения).
Flags
Комбинация флагов (на данный момент флагов нет, установите в ноль).
Count
Количество элементов, возвращённых скриптом (значение устанавливает Far Manager).
Values
Указатель на массив элементов, возвращённых скриптом (значение устанавливает Far Manager).

Замечания

  1. Скрипт SequenceText не должен вызывать функции, перечисленные в статье "Ограничения в использовании некоторых функций" в macroapi_manual.chm.
  2. Массив Values может использоваться плагином только до следующего вызова функции MacroControl, либо до возвращения управления Far Manager (раннее из этих событий). Если предполагается использовать элементы массива и далее, плагин должен скопировать их в свою память.
