FarSettingsEnum

главная | структуры
Структура FarSettingsEnum используется в функции SettingsControl для управления настройками. 
struct FarSettingsCreate
{
  size_t Root;
  size_t Count;
  const struct FarSettingsName *Items;
};

Элементы

Root
Описатель ключа, откуда брать информацию.
Count
Количество элементов.
Items
Массив элементов типа FarSettingsName.
