GetVirtualFindDataInfo

главная | структуры
Структура GetVirtualFindDataInfo передает информацию Far Manager'у о списке файлов из эмулируемой файловой системы в функции GetVirtualFindDataW. 
struct GetVirtualFindDataInfo
{
  size_t StructSize;
  HANDLE hPanel;
  struct PluginPanelItem *PanelItem;
  size_t ItemsNumber;
  const wchar_t *Path;
};

Элементы

StructSize
Это поле должно содержать размер структуры GetVirtualFindDataInfo.
hPanel
Описатель панели, возвращаемый функцией OpenW. Поле заполнено Far Manager'ом.
PanelItem
Переменная, которая получит адрес массива структур PluginPanelItem.
ItemsNumber
Переменная, которая получит число структур PluginPanelItem.
Path
Путь, для которого запрашивается список файлов (текущий каталог на другой панели). Путь должен быть завершён обратным слешем ("\").
