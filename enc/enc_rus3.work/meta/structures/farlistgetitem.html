FarListGetItem

главная | структуры | Dialog API
Структура FarListGetItem описывает один элемент типа FarListItem для элементов диалога DI_LISTBOX и DI_COMBOBOX. 
struct FarListGetItem
{
  int ItemIndex;
  struct FarListItem Item;
};

Элементы

ItemIndex
Индекс элемента из списка.
Item
Элемент списка типа FarListItem.
