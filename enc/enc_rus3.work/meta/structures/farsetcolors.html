FarSetColors

главная | структуры
Структура FarSetColors используется в функции AdvControl для изменения цветовой схемы Far Manager. 
struct FarSetColors
{
  FARCOLORFLAGS Flags;
  size_t StartIndex;
  size_t ColorItem;
  struct FarColor *Colors;
};

Элементы

Flags
Комбинация флагов (перечисление FARSETCOLORFLAGS):
ФлагОписание
FSETCLR_REDRAW после изменений цветовой схемы обновить экран
StartIndex
Начальная позиция индекса цветовой схемы для замены.
ColorItem
Количество позиций цветовой для замены.
Colors
Указатель на массив цветовой схемы - массив структур FarColor цветовых атрибутов.
