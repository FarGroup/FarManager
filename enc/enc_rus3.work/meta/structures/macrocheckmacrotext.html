Структура MacroCheckMacroText используется в функции MacroControl для проверки макропоследовательности. 
struct MacroCheckMacroText
{
  union
  {
    struct MacroSendMacroText Text;
    struct MacroParseResult   Result;
  }
#ifndef __cplusplus
  Check
#endif
  ;
};

Элементы

Text
Поле содержит макропоследовательность, структура MacroSendMacroText, которую необходимо проверить.
Result
Результат проверки макропоследовательности, структура MacroParseResult. Содержимое поля Text затирается.
