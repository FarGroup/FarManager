FarDialogItemColors

главная | структуры | Dialog API
Структура FarDialogItemColors используется в Диалог АПИ для изменения цветовых атрибутов элементов. 
struct FarDialogItemColors
{
  size_t StructSize;
  unsigned __int64 Flags;
  size_t ColorsCount;
  struct FarColor* Colors;
  void* Reserved;
};

Элементы

StructSize
Размер структуры.
Flags
Флаги, поле используется плагином.
ColorsCount
Количество элементов в массиве Colors.
Colors
Установленный или устанавливаемый цвет, структура типа FarColor.
Reserved
Поле используется плагином.
