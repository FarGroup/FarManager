Common API

главная

Экспортируемые функции, т.е. те которые плагины экспортируют для Far Manager

ФункцияОписание
ExitFARW перед выходом из Far Manager
OpenW вызывает плагин
ProcessConsoleInputW предобработка консольных событий плагином
ProcessSynchroEventW позволяет плагину выполнить код в контексте главного потока Far Manager
SetStartupInfoW Far Manager передает плагину необходимую информацию

Сервисные функции, указатели на эти функции плагины получают от Far Manager в функции SetStartupInfoW

ФункцияОписание
AdvControl дополнительные управляющие функции; работают везде: в панелях/редакторе/вьюере
ColorDialog диалог изменения цвета
GetMsg получить строку сообщения из языкового файла
InputBox позволяет ввести строку текста
Menu показывает меню
Message показывает сообщение
RegExpControl управляющие функции Регулярных выражений
RestoreScreen восстанавливает предварительно сохранённую область экрана
SaveScreen сохраняет область экрана
ShowHelp показывает нужный топик из hlf-файла
Text выводит текстовую строку на экран

Структуры

СтруктураОписание
ExitInfo при закрытии плагина
FAR_CHAR_INFO экранная символьная ячейка
FarColor код цвета
FarKey код клавиши
FarMenuItem элемент меню
FarSetColors изменение цветовой схемы Far Manager
OpenInfo информация откуда открыт плагин
PluginStartupInfo различная важная информация
ProcessConsoleInputInfo информация о консольном событии Far Manager
ProcessSynchroEventInfo информация о событии при синхронизации потоков
ProgressValue корректировка прогресса операции на кнопке Far Manager
RegExpMatch используется в структуре RegExpSearch
RegExpSearch используется в Регулярных выражениях Far Manager
WindowInfo информация об окне Far Manager
WindowType информация о типе окне Far Manager
