ProcessDialogEventW

главная | экспортируемые функции | Dialog API
Функция ProcessDialogEventW сообщает плагину о событиях, происходящих в диалоге Far Manager. 
int WINAPI ProcessDialogEventW(
  const struct ProcessDialogEventInfo *Info
);

Параметры

Info
Указатель на структуру ProcessDialogEventInfo, Far Manager передает информацию о событии.
В начале функции проверьте, что поле ProcessDialogEventInfo.StructSize >= sizeof(ProcessDialogEventInfo).

Возвращаемое значение

TRUE - плагин самостоятельно обработал событие. Переменная ProcessDialogEventInfo.FarDialogEvent.Result будет использована в качестве возвращаемого значения обработчика диалога.
FALSE - продолжить обработку события.
