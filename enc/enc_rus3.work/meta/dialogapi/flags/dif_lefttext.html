DIF_LEFTTEXT

главная | Dialog API | Флаги элементов диалога

Флаг DIF_LEFTTEXT позволяет выравнивать заголовки рамок по левому краю. По умолчанию заголовки рамок выровнены по-центру.

Элементы

Флаг DIF_LEFTTEXT имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_DOUBLEBOX двойная рамка
DI_SINGLEBOX одиночная рамка
