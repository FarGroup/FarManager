DIF_CENTERTEXT

главная | Dialog API | Флаги элементов диалога

Флаг DIF_CENTERTEXT позволяет выравнивать текст в элементах DI_TEXT и DI_VTEXT по центру, относительно геометрии элемента.

Элементы

Флаг DIF_CENTERTEXT имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_TEXT Текстовая строка.
DI_VTEXT Вертикальная текстовая строка.

Замечания

При указании этого флага вы должны обязательно корректно заполнить координаты X2 (для DI_TEXT) и Y2 (для DI_VTEXT).
