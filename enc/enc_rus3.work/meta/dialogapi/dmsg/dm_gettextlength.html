DM_GETTEXTLENGTH

главная | сообщения | Dialog API
Сообщение DM_GETTEXTLENGTH посылается Менеджеру Диалогов для получения размера данных, содержащихся в FarDialogItem.Data.

Param1

ID элемента диалога.

Param2

0

Return

Размер данных в символах, без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Все элементы диалога.
