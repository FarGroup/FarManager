DN_ACTIVATEAPP

главная | события | Dialog API
Событие DN_ACTIVATEAPP приходит в обработчик диалога, когда консольное окно Far Manager становится активным или пассивным.

Param1

1 - консольное окно получило фокус ввода (стало активным) 0 - консольное окно потеряло фокус ввода (стало пассивным)

Param2

0

Return

0
