DM_SETDLGITEM

Сообщения | Dialog API
Сообщение DM_SETDLGITEM посылается Менеджеру диалогов для изменения информации о заданном элементе.

Param1

ID элемента диалога

Param2

Указатель на заполненную структуру FarDialogItem

Return

TRUE - данные обработаны. FALSE - если такого ID элемента не существует или вы поменяли тип элемента.

Элементы

ЭлементОписание
Все Все элементы диалога

Замечания

Вы не можете изменить тип элемента.
