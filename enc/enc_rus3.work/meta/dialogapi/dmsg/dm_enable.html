DM_ENABLE

Сообщения | Dialog API
Команда DM_ENABLE посылается ядру Dialog API для переключения или получения состояния Enable/Disable конкретного элемента диалога.

Param1

ID элемента диалога

Param2

Состояние элемента:
TRUE - Enable
FALSE - Disable
-1 - получить состояние

Return

Предыдущее состояние элемента:
TRUE - Enabled
FALSE - Disabled

Элементы

ЭлементОписание
Все Все элементы диалога
