DN_CLOSE

События | Dialog API
Событие DN_CLOSE приходит в обработчик диалога как сигнал, сообщающий, что диалог будет закрыт - пользователь пытается завершить работу с диалогом.

Param1

  • ID элемента, который имел фокус ввода в момент нажатия Ctrl-Enter или ID элемента, у которого поле DefaultButton = 1.
  • -2 - нажали KEY_BREAK (комбинация клавиш Ctrl-Pause или Ctrl-ScrollLock)
  • -1 - KEY_ESC или KEY_F10

Param2

0

Return

TRUE - да, согласен с выходом из диалога
FALSE - продолжить работу диалога.

Замечания

  1. Событие DN_CLOSE приходит сразу после посылки сообщения DM_CLOSE обработчиком диалога.
