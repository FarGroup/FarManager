DM_SETITEMDATA

Сообщения | Dialog API
Сообщение DM_SETITEMDATA устанавливает 32-битное значение, связанное с элементом диалога. Каждый элемент диалога имеет соответствующее 32-битное значение, предназначенное для использования плагином в его собственных целях.

Param1

ID элемента управления.

Param2

Новое 32-битное значение.

Return

Предыдущее значение.
