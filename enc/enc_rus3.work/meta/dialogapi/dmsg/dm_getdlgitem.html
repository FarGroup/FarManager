DM_GETDLGITEM

Сообщения | Dialog API
Сообщение DM_GETDLGITEM посылается Менеджеру диалогов для получения полной информации о заданном элементе.

Param1

ID элемента диалога

Param2

Указатель на буфер, структуру FarDialogItem, память плагин выделят самостоятельно.

Return

Необходимый размер буфера, при Param2 = NULL.

Элементы

ЭлементОписание
Все Все элементы диалога

Замечания

Пример

// получить 12-й элемент
// выделяем память
struct FarDialogItem *DialogItem = (FarDialogItem *) malloc(Info.SendDlgMessage(hDlg,DM_GETDLGITEM,12,NULL));

// получаем элемент
Info.SendDlgMessage(hDlg,DM_GETDLGITEM,12,(LONG_PTR)DialogItem);

// что-то делаем

// освобождаем память 
free(DialogItem);
