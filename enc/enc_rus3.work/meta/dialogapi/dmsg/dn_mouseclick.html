DN_MOUSECLICK

События | Dialog API
Событие DN_MOUSECLICK посылается обработчику после того, как пользователь кликнул клавишей мыши на одном из элементов диалога или вне диалога.

Param1

ID элемента диалога.
Param1 = -1 для случая, если пользователь кликнул клавишей мыши вне диалога.

Param2

Указатель на MOUSE_EVENT_RECORD.

Return

TRUE - обработчик диалога сам обработал событие, дальше ядру не продолжать
FALSE - продолжить обработку этого события ядром.

Элементы

ЭлементОписание
Все Все элементы диалога, кроме скрытых и недоступных.

Замечания

  1. Для элемента DI_USERCONTROL координаты курсора (0,0) мыши начинаются с левого-верхнего угла этого элемента. Для остальных элементов - относительно левого верхнего угла экрана.
  2. Обработчик диалога может отловить событие DOUBLE_CLICK. Для этого необходимо проверить флаг MouseEvent.dwEventFlags==DOUBLE_CLICK
  3. Если пользователь кликнул клавишей мыши вне диалога, то:
    • левая клавиша рассматривается как нажатие клавиши <Esc>;
    • правая клавиша рассматривается как нажатие клавиши <Enter>;
    • прочие клавиши (для многокнопочной мыши) не имеют никакого эффекта.
    • если функция диалога не обрабатывает событие DN_MOUSECLICK с Param1=-1, то диалог будет закрыт
