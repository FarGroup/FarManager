DialogInfo

главная | Dialog API
Структура DialogInfo позволяет узнать информацию о диалоге, является параметром в DM_GETDIALOGINFO. 
struct DialogInfo
{
  size_t StructSize;
  GUID Id;
  GUID Owner;
};

Элементы

StructSize
Размер структуры.
Id
Уникальный идентификатор диалога.
Owner
Поле содержит GUID плагина, создавшего диалог.
