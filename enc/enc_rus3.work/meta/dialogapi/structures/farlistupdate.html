FarListUpdate

главная | Dialog API
Структура FarListUpdate задаёт параметры операции обновления элемента в списке DI_LISTBOX или DI_COMBOBOX: индекс обновляемого элемента и новые данные элемента. 
struct FarListUpdate
{
  int Index;
  struct FarListItem Item;
};

Элементы

Index
Индекс элемента в списке.
Items
Элемент FarListItem.

Замечания

Данные ассоциированные с пунктом списка при обновлении не уничтожаются. Для автоматического уничтожения копии данных используйте флаг LIF_DELETEUSERDATA.
