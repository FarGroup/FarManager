SendDlgMessage

главная | Dialog API Сообщения
Функция SendDlgMessage позволяет послать сообщение функции-обработчику диалога. 
INT_PTR WINAPI SendDlgMessage(
  HANDLE hDlg,
  int    Msg,
  int    Param1,
  void  *Param2
);

Параметры

hDlg
Указатель на диалог
Msg
Одно из сообщений DM_*
Param1
Параметр 1
Param2
Параметр 2

Возвращаемое значение

Значение, возвращаемое функцией SendDlgMessage зависит от параметра Msg.

Замечания

Фрагмент обработчика диалога игры Reversi:
