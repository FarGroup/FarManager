LStricmp

главная | стандартные функции
Функция FSF.LStricmp сравнивает две строки без учёта регистра. 
int WINAPI LStricmp(
  const wchar_t *Str1,
  const wchar_t *Str2
);

Параметры

Str1, Str2
Указатели на строки, которые вы хотите сравнить.

Возвращаемое значение

Функция возвращает:
-1 - если s1 < s2
1 - если s1 > s2
0 - если s1 == s2
