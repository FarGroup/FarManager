Plugins' features overview

main | Internet

The FAR Manager shell is so tightly integrated with its plugins that there's just no sense to talk about FAR and not to mention them. Plugins vastly expand the FAR features while supplementing and modifying them.

Without going into details, some of the features can be noted:

  • printers control, both local and network ones
  • syntax highlighting in the program source texts
  • working with FTP-servers (including access through various proxy types, automatic download resume, etc.)
  • search and replace in many files simultaneously, using regular expressions
  • renaming groups of files with support for compound masks consisting of substitution symbols and templates
  • NNTP/SMTP/POP3/IMAP4 clients and sending messages to a pager
  • working under non-standard console screen resolutions
  • texts conversion from one national code page to another
  • Recycle Bin contents handling
  • process priority control on local or network computer
  • words autocompleting in editor and templates handling
  • Windows registry editing
  • Windows shortcuts creation and modification
  • file and text operations that make FidoNet using more comfortable
  • files UU-encode and UU-decode
  • WinAmp control and MP3 tags modification
  • Quake PAK-files processing
  • working with dababases using ODBC (and OCI for Oracle)
  • RAS service control
  • external programs execution (compilers, converters, etc.) while editing text in FAR editor
  • Windows help files contents displaying (.hlp and .chm)
  • calculators with different possibilities
  • various games :-)
  • spell-check while editing text in FAR editor
  • removable drives catalog preparation and many more...
