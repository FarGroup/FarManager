LIsAlphanum

main | FarStandardFunctions
The FSF.LIsAlphanum function tests whether the given character is a letter or a number. This function works in OEM code page. 
int WINAPI LIsAlphanum(
  unsigned Ch
);

Parameters

Ch
The character you want to test.

Return value

If the given character is a letter or a number returns TRUE, otherwise returns FALSE.
