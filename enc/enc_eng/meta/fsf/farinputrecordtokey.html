FarInputRecordToKey

main | FarStandardFunctions
The FSF.FarInputRecordToKey function is used to convert a key code from an INPUT_RECORD structure to an internal FAR key code. 
int WINAPI FarInputRecordToKey(
  INPUT_RECORD *Rec
);

Parameters

Rec
Pointer to an INPUT_RECORD structure you want to convert.

Return value

Return value is an internal FAR key code.
