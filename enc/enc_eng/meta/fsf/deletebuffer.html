DeleteBuffer

main | FarStandardFunctions
The FSF.DeleteBuffer function is used to free an allocated buffer returned by the PasteFromClipboard function. 
void WINAPI DeleteBuffer(
  void *Buffer
);

Parameters

Buffer
Pointer to the buffer that needs to be freed.

Return value

None.

Remarks

This function must be used to free FAR memory blocks since plugin's memory manager can be different from the one used in FAR.
