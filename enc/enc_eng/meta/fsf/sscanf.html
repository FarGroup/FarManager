sscanf

main | FarStandardFunctions
The FSF.sscanf function allows to read formatted data from a string. 
int WINAPI sscanf(
  const char *Buffer,
  const char *Format,
  [address, ...]
);

Parameters

Buffer
Buffer that will be scanned.
Format
Format string
address
Series of arguments that receive data in accordance with the formatted string.

Return value

If the function succeeds the return value is the number of fields successfully converted and assigned. -1 is returned if the number of format specifiers is greater than the number of fields in the scanned string. On error return value is 0.

Remarks

See a C/C++ run-time library reference for more information.
