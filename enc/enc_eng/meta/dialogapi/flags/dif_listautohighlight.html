DIF_LISTAUTOHIGHLIGHT

main | Dialog API | Dialog item flags

If the DIF_LISTAUTOHIGHLIGHT flag is set, then hot keys will be assigned automatically starting from the first item.

Controls

The DIF_LISTAUTOHIGHLIGHT flag is applicable to the following dialog items:
ControlDescription
DI_LISTBOX List box.
DI_COMBOBOX Combo box.
