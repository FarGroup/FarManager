DIF_VAREDIT

main | Dialog API | Dialog item flags

The DIF_VAREDIT flag is the "512 bytes barrier overcoming for an edit box". The Dialog Manager, while working with an edit box or a combo box (with DIF_VAREDIT flag set) will take into account only Ptr.* members of the FarDialogItem structure.

Controls

The DIF_VAREDIT flag is applicable to the following dialog items:
ControlDescription
DI_COMBOBOX Combo box.
DI_EDIT Edit box.

Remarks

  1. The plugin itself must take care of memory allocation for FarDialogItem.Ptr.PtrData and fill in the size of this buffer correctly (FarDialogItem.Ptr.PtrLength).

  2. If the size of the data being used does not exceed 512 bytes (or sizeof(FarDialogItem.Data)), then there is no sence in using the DIF_VAREDIT flag.
