DIF_LISTWRAPMODE

main | Dialog API | Dialog item flags

If the DIF_LISTWRAPMODE flag is set, then attempts to move the cursor up from the first item or down from the last item will result in movement to the last or the first item, respectively.

Controls

The DIF_LISTWRAPMODE flag is applicable to the following dialog items:
ControlDescription
DI_LISTBOX List box.
DI_COMBOBOX Combo box.
