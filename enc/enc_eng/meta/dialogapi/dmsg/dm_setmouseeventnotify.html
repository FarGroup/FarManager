DM_SETMOUSEEVENTNOTIFY

Messages | Dialog API
The DM_SETMOUSEEVENTNOTIFY message allows to control the mechanism for sending initial non-altered mouse event (DN_MOUSEEVENT) to a dialog manager before they are processed by the dialog kernel.

Param1

0 - off
1 - on
-1 - get state

Param2

0

Return

Previous state.

Controls

ControlDescription
Dialog message applies to the dialog only
