DN_LISTCHANGE

Events | Dialog API
The DN_LISTCHANGE event is sent to the dialog callback function to notify of changes that occurred to an item in the list of the DI_COMBOBOX, DI_LISTBOX or DIF_HISTORY types.

Param1

Dialog item ID.

Param2

Current position in the list.

Return

TRUE - allow the changes
FALSE - discard changes.

Controls

ControlDescription
DI_COMBOBOX Combined list
DI_LISTBOX List box
DI_EDIT Text input control (with the DIF_HISTORY flag set)
DI_FIXEDIT Fixed size text input control (with the DIF_HISTORY flag set)
