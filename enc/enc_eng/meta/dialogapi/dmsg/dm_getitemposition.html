DM_GETITEMPOSITION

Messages | Dialog API
The DM_GETITEMPOSITION message is sent to the dialog manager to retrieve the size and position of a dialog item.

Param1

The ID of the dialog item for which the position is retrieved.

Param2

Pointer to a SMALL_RECT structure that will contain the size and position of the item.

Return

TRUE if the data has been copied or FALSE if Param2 is NULL or the item does not exist.
