DM_USER

Messages | Dialog API
The DM_USER message is intended to specify the starting value for user-defined messages.

Param1

User-defined value.

Param2

User-defined value.

Return

User-defined value.

Controls

ControlDescription
All All dialog items

Remarks

Example

#define DMREV_RESULT DM_USER+1

LONG_PTR WINAPI ReversiDialogProc(HANDLE hDlg, int Msg,int Param1,LONG_PTR Param2)
{
...
    Info.SendDlgMessage(hDlg,DMREV_RESULT,0,0);
...
    case DMREV_RESULT: // user's definition
      ...
      return 0;
...
}
