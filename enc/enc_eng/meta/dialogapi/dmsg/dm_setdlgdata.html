DM_SETDLGDATA

Messages | Dialog API
The DM_SETDLGDATA message is sent to the dialog manager to set the 32-bit data value associated with the dialog. This value is designed for use by the plugin that created the dialog.

Param1

0

Param2

New 32-bit value.

Return

Previous value.

Remarks

Initially the Param value of DialogEx is here.
