DM_RESIZEDIALOG

Messages | Dialog API
The DM_RESIZEDIALOG message is sent to the dialog manager to resize the dialog window.

Param1

0

Param2

Pointer to a COORD structure containing the new size of the dialog window.

Return

The COORD structure containing the new size of the dialog window.
