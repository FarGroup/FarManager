DM_SETITEMPOSITION

Messages | Dialog API
The DM_SETITEMPOSITION message is sent to the dialog manager to change the position of a dialog item.

Param1

The ID of the dialog item for which the position is changed.

Param2

Pointer to a SMALL_RECT structure containing the new item coordinates.

Return

TRUE - item position has been changed.
FALSE - item wiht such ID does not exist.

Controls

ControlDescription
All All dialog items
