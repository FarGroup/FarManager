DM_LISTFINDSTRING

Messages | Dialog API
The DM_LISTFINDSTRING message is sent to the dialog manager to find an element by pattern in a DI_COMBOBOX or DI_LISTBOX list.

Param1

The ID of the dialog item in which the item is searched.

Param2

Pointer to a FarListFindstructure specifying the element to find.

Return

Index of the found element (0-based), or -1 if the specified item was not found.

Controls

ControlDescription
DI_LISTBOX list
DI_COMBOBOX combined list
