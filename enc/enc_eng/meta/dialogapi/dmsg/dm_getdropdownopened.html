DM_GETDROPDOWNOPENED

Messages | Dialog API
The DM_GETDROPDOWNOPENED message is sent to the dialog manager to determine: "if there is an open combo box or history list in the dialog".

Param1

0

Param2

0

Return

FALSE - there is no open combo box or history list.
TRUE - there is an open combo box or history list.

Controls

ControlDescription
Dialog message concerning dialog only
