DM_GETDLGDATA

Messages | Dialog API
The DM_GETDLGDATA returns to the dialog handler a 32-bit data value associated with the dialog. Each dialog has a corresponding 32-bit value designed for use by the plugin that created the dialog.

Param1

0

Param2

0

Return

The dialog manager returns 32-bit value passed by plugin earlier.

Remarks

Initially the 32-bit data value contains the data passed in Param to the DialogEx function.
