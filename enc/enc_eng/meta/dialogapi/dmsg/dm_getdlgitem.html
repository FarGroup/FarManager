DM_GETDLGITEM

Messages | Dialog API
Сообщение DM_GETDLGITEM посылается Менеджеру диалогов для получения полной информации о заданном элементе.

Param1

ID элемента диалога

Param2

Указатель на структуру FarDialogItem.

Return

TRUE - данные скопированы или FALSE - для неверного ID элемента.

Controls

ControlDescription
Все Все элементы диалога

Remarks

Example

// получить параметры 12-го элемента
struct FarDialogItem DialogItem;

Info.SendDlgMessage(hDlg,DM_GETDLGITEM,12,(long)&DialogItem);
