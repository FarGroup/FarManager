DM_GETCHECK

Messages | Dialog API
The DM_GETCHECK message is sent to the dialog manager to retrieve the state of DI_CHECKBOX or DI_RADIOBUTTON items.

Param1

The ID of the dialog item for which you want to retrieve the state.

Param2

0

Return

State of the item: BSTATE_UNCHECKED, BSTATE_CHECKED, BSTATE_3STATE.

Controls

ControlDescription
DI_CHECKBOX Check Box
DI_RADIOBUTTON Radio Button
