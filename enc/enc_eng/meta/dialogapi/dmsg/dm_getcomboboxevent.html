DM_GETCOMBOBOXEVENT

Messages | Dialog API
The DM_GETCOMBOBOXEVENT message allows to determine if the sending of DN_KEY or DN_MOUSEEVENT events for an open DI_COMBOBOX is enabled or disabled.

Param1

ID of a DI_COMBOBOX.

Param2

0

Return

State flags (FARCOMBOBOXEVENTTYPE enum).

Controls

ControlDescription
DI_COMBOBOX combined list
