DM_SETITEMDATA

Messages | Dialog API
The DM_SETITEMDATAmessage is sent to the dialog manager to set the 32-bit data value associated with a dialog item. Every dialog item has a corresponding 32-bit data value designed for use by the plugin that created the dialog.

Param1

The ID of the dialog item for which the data is retrieved.

Param2

The new 32-bit data value.

Return

Previous data value.
