FarListTitles

main | structures | Dialog API
The FarListTitles structure specifies the titles (top and bottom) of a DI_LISTBOX list. 
struct FarListTitles
{
  int   TitleLen;
  char *Title;
  int   BottomLen;
  char *Bottom;
};

Members

TitleLen
Length of the top title string.
Title
Top title string. May be NULL; in this case, the title is not shown.
BottomLen
Length of the bottom title string.
Bottom
Bottom title string. May be NULL; in this case, the title is not shown.

Remarks

TitleLen and BottomLen members are required only while getting list titles (see DM_LISTGETTITLES)
