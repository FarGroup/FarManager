FarListGetItem

main | Dialog API
The FarListGetItem structure describes one item of a FarListItem structure for DI_LISTBOX and DI_COMBOBOX dialog items. 
struct FarListGetItem
{
  int ItemIndex;
  struct FarListItem Item;
};
