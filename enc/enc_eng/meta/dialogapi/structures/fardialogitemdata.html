FarDialogItemData

main | Dialog API
The FarDialogItemData structure describes data, being sent (received) to the dialog item by a DM_SETTEXT message (DM_GETTEXT) as Param2 parameter. 
struct FarDialogItemData
{
  int   PtrLength;
  char *PtrData;
};

Members

PtrLength
Data size in PtrData without ending NULL character.
PtrData
Pointer to the data being sent.
