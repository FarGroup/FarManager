FarList

main | Dialog API
The FarList structure contains a pointer to an array of FarListItem structures for DI_LISTBOX or DI_COMBOBOX controls. 
struct FarList
{
  DWORD ItemsNumber;
  struct FarListItem *Items;
};

Members

ItemsNumber
The number of list items.
Items
Pointer to an array of FarListItem structures containing a list of combobox, dropdown list or list box items.
