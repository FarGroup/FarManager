SMatches

main | Custom API | regexp
Returns the last compilation result. 
#define MATCHESNUM 0x10
typedef struct SMatches
{
  int s[MATCHESNUM];
  int e[MATCHESNUM];
  int CurMatch;
} *PMatches;

Members

s
array of matched opening bracket offsets
e
array of matched closing bracket offsets
CurMatch
Total number of brackets in the regexp

Remarks

If there's no match for a bracket (it was skipped) then both s and e are -1. Mind that CurMatch is not necessary equals to the number of brackets matched - the latter can occur in any order.
