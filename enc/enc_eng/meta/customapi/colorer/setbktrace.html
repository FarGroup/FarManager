CRegExp.SetBkTrace

main | Custom API | regexp
Set references for \yN operator

Class method:

bool SetBkTrace(char *Str, PMatches Trace);

DLL interface:

BOOL WINAPI reSetBkTrace(PRegExp re, char *str, PMatches trace);

Members

re
a pointer to an object
Str
string for setting a relation
trace
set of bracket matches for the previous regexp

Return value

Returns TRUE or FALSE depending of operation success/failure.

Remarks

This function is used for setting a relation between different regexps (it's used in the colorer plugin, for example) - by means of \yN operator, where N - bracket number in the first regexp for reference.
