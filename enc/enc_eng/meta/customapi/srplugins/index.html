Search and Replace plugins programming

Custom API

Search and Replace (S&R) is a FAR plugin that allows to search and replace information both in single file and in multiple ones. It also works in the viewer and the editor.

SRP-modules API information (help file srplugins.chm is needed) is available from the plugin's author: Ivan Sinturin http://www.moris.ru/~spinoza. Also the on-line version is available here: http://www.moris.ru/~spinoza/download/s_and_r/srplugins
