LoadFormatModule

main | Custom API | archive support
It's called when a subplugin is finished loading. This function is optional and can be omitted. 
DWORD WINAPI LoadFormatModule(
  const char *ModuleName
);

Parameters

ModuleName
subplugin module name (FMT file full path name)

Return value

Must be 0. May be used in the future for returning information about a subplugin.

Remarks

This function is called only once when MultiArc loads a subplugin (see MultiArc sources - ARCPLG.CPP file, ArcPlugins::LoadFmtModules function).
