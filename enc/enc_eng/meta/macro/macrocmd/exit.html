$Exit

main | Macros | Macro-language | Macro-commands

ATTENTION!

  • It is impossible to use macro-language elements while recording a macro in a usual way. Macro-language elements can be added to the sequence only by editing the registry manually or by using special applications or FAR plugins.
The $Exit macro-command stops the execution of a macro sequence. 
$Exit

Arguments

None.

Example

REGEDIT4

;remove the selected characters from the command line

[HKEY_CURRENT_USER\Software\Far\KeyMacros\Common\ShiftDel]
"Sequence"="$If (Qview || Shell)
$If (!APanel.Visible) ShiftDel $Exit $End CtrlO ShiftDel CtrlO
$Else ShiftDel $End"
"DisableOutput"=dword:00000001
"NotEmptyCommandLine"=dword:00000001
