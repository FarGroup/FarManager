$While

main | Macros | Macro-language | Macro-commands
The $While loop macro-command macro-command executes the Sequence of the macro-commands while the Expr is true. 
$While (Expr) Sequence $End

Elements

Expr
Loop conditional expression.
Sequence
Sequence to be executed while Expr is true.

Notes

  1. It is impossible to use macro-language elements while recording a macro in a usual way. Macro-language elements can be added to the sequence only by editing the registry manually or by using special applications or FAR plugins.

Example

REGEDIT

;set the cursor to the nearest file in the panel

[HKEY_CURRENT_USER\Software\Far\KeyMacros\Shell\AltAdd]
"Sequence"="$While (APanel.Folder && !APanel.Eof) Down $End"
"DisableOutput"=dword:00000001
