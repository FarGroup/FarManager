PPanel.Bof		 true if the current panel item is the first

PPanel.Eof		 true if the current panel item is the last

PPanel.Empty		 true if the panel is empty

PPanel.LFN		 true if LFN (long file names) is enabled for the panel

command line object states
CmdLine.Bof true if the cursor is at the beginning of the command line
CmdLine.Eof true if the cursor is at the end of the command line
CmdLine.Empty true if the command line is empty
CmdLine.Selected true if there is a selected block of text within the command line
execution context dependent object states
Bof достигнуто начало файла в редакторе, программе просмотра; в панелях типа QView/Info/Tree - достигнуто начала файла в области просмотра, в прочих "панелях" - курсор в начале командной строки; в строках ввода в диалогах - в начале строки; в списках/меню - указатель установлен на первом пункте.
Eof достигнут конец файла в редакторе, программе просмотра; в панелях типа QView/Info/Tree - достигнут конец файла в области просмотра, в прочих "панелях" - курсор в конце командной строки; в строках ввода в диалогах - в конце строки; в списках/меню - указатель установлен на последнем пункте
Empty в панелях типа QView/Info/Tree - область просмотра пуста, иначе - командная строка пуста; строка ввода пуста; в редакторе/программе просмотра пустой файл; список пуст.
Selected выделен блок в строках ввода, в редакторе или программе просмотра, в панелях типа QView/Info/Tree - выделен блок в области просмотра, в прочих "панелях" - выделен блок в командной строке (текущая область Shell).
IClip true if FAR Manager is working with the internal clipboard (See '$IClip')
Windowed true if FAR Manager is in the windowed mode
state of macro command parameters
DisableOutput true if the screen output is disabled

Notes

  1. It is impossible to use macro-language elements while recording a macro in a usual way. Macro-language elements can be added to the sequence only by editing the registry manually or by using special applications or FAR plugins.
