Макроязык

main | Macros | Макроязык

В макроязыке используются специальные макрокоманды.

ATTENTION!

  • Элементы макроязыка невозможно использовать во время записи макроса стандартным образом. Добавить элементы языка в макрос можно только путем ручной правки реестра или путем применения специальных программ/плагинов.

Команды, используемые в тексте макросов

Макрокоманда Description
$Date ["format"] Предназначена для вставки текущей даты/времени.
Подробнее см. раздел "$Date".
$Text "string" Предназначена для вставки произвольного текста.
Подробнее см. раздел "$Text".
$IClip Работа с буфером обмена.
Подробнее см. раздел "$IClip".
$MMode 1 Позволяет изменять режим перерисовки экрана во время проигрывания макропоследовательности.
Подробнее см. раздел "$MMode".
$Exit Прерывает выполнение макропоследовательности.
Подробнее см. раздел "$Exit".
$XLat Вызывает функцию транслитерации.
Подробнее см. раздел "$XLat".
$KbdSwitch Циклическое переключение раскладки клавиатуры.
Подробнее см. раздел "$KbdSwitch".
$If (Expr) TruePart [$Else FalsePart] $End Позволяет организовать в макросах процесс ветвления макропоследовательности.
Подробнее см. раздел "$If".
$While (Expr) Sequence $End Позволяет организовать в макросах циклический процесс - "Пока условие Expr истинно выполнять последовательность Sequence".
Подробнее см. раздел "$While".
$Rep (Expr) Sequence $End Позволяет организовать в макросах циклический процесс - "Выполнить Expr раз последовательность Sequence".
Подробнее см. раздел "$Rep".
%var=value;
%%var=value;		 Использование переменных.
Подробнее см. раздел "Переменные".
