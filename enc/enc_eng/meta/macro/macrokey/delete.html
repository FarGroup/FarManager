Deleting a macro

main | Macros | Using macros

To delete a previously recorded macro you should record an empty (with no actions) macro and assign it the same shortcut key combination as in the existing one that you would like to remove.

You can do it as following:

  1. Start recording a macro (Ctrl-<.>)
  2. Stop recording a macro (Ctrl-<.>)
  3. Enter or select in the dropdown box the shortcut key combination value of the macro command you want to remove.

ATTENTION!

  • Shortcut key combination will perform its original functions if any as soon as you delete the macro command. I.e. if some macro command overrides some function performed by a plugin or FAR manager then you will return this function back to a plugin module or FAR manager by deleting this macro.
