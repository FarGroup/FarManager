Macros

main

Macro command or 'macro' in FAR is a recorded sequence of keystrokes that can be assigned to a shortcut command. Macro can be used to repeatedly perform a certain sequence of user actions at a time by pressing a single hotkey.

Generally macros can be used to:

  1. Assign an often used set of actions to a shortcut to use it repeatedly;
  2. Perform special functions which can be specified in a macro command text by using special commands;
  3. Redefine the standard FAR hotkeys.

Mostly macros are used to define shortcut access to the external plugin modules and redefine hotkeys for standard FAR functions.
