EditorConvertText

main | structures
The EditorConvertText structure is used in the EditorControl function for text conversion from OEM-encoding to the current FAR editor's codepage (and vice versa). 
struct EditorConvertText
{
  char *Text;
  int TextLength;
};

Elements

Text
Points to text for conversion.
TextLength
Length of text for conversion.
