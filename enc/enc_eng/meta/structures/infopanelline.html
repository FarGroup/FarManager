InfoPanelLine

main | structures
The InfoPanelLine structure describes a single line n the information panel. An array of InfoPanelLine structures is passed to FAR by the GetOpenPluginInfo function. 
struct InfoPanelLine
{
  char Text[80];
  char Data[80];
  int Separator;
};

Elements

Text
Parameter header - left-aligned text displayed using the regular text color (COL_PANELTEXT).
Data
Parameter data - right-aligned text displayed using the selected text color (COL_PANELINFOTEXT)
Separator
If non-zero, a separator is displayed. Text is used as the separator header, Data is ignored.
