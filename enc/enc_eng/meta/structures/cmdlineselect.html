CmdLineSelect

main | structures
The CmdLineSelect structure is used in the Control function for text selection/deselection on the FAR command line, or getting selection information thereof. 
struct CmdLineSelect
{
  int SelStart;
  int SelEnd;
};

Elements

SelStart
Selection start position.
SelEnd
Selection end position.

Remarks

To clear the selection, set SelStart and SelEnd to -1.
