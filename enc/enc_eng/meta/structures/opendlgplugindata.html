OpenDlgPluginData

main | structures
Pointer to an OpenDlgPluginData structure is passed to an OpenPlugin function, when plugin is called from dialog. 
struct OpenDlgPluginData
{
  int ItemNumber;
  HANDLE hDlg;
};

Elements

ItemNumber
Position of the activated plugin item in the exported items list in plugins menu.
hDlg
Dialog handle.
