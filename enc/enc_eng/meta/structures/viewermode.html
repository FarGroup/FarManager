ViewerMode

main | structures
The ViewerMode structure is used to recieve additional information about the state of the current viewer instance. 
struct ViewerMode
{
   int UseDecodeTable;
   int TableNum;
   int AnsiMode;
   int Unicode;
   int Wrap;
   int TypeWrap;
   int Hex;
   DWORD Reserved[4];
};

Elements

UseDecodeTable
If TRUE, then a decoding table is used.
TableNum
The number of the user character table. Use only if UseDecodeTable is TRUE.
AnsiMode
If TRUE, then the current charset is ANSI.
Unicode
If TRUE, then the current charset is Unicode.
Wrap
Text wrapping is on - TRUE, off - FALSE.
TypeWrap
Text wrapping mode.
Hex
If TRUE, then the viewer is in hex-mode.
Reserved
Reserved for future use.
