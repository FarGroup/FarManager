EditorSaveFile

main | structures
The EditorSaveFile structure is used in the EditorControl function to save the currently edited file (the ECTL_SAVEFILE command). 
struct EditorSaveFile
{
  char FileName[NM];
  char *FileEOL;
};

Elements

FileName
Optional new name for the file (full path required). Set the first byte of the FileName field to zero to retain the current file name.
FileEOL
End-of-line sequence. Can be NULL (leave current EOL-sequence), \r\n or \n
