EditorConvertPos

main | structures
The EditorConvertPos structure is used in the EditorControl function to convert a position in the line to a screen position (and vice versa). This is useful for lines which contain tab characters. 
struct EditorConvertPos
{
  int StringNumber;
  int SrcPos;
  int DestPos;
};

Elements

StringNumber
Line number or -1 for current line.
SrcPos
Source position.
DestPos
Converted position.
