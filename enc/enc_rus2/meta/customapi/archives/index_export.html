Экспортируемые функции FMT-модулей

главная | Custom API | поддержка архивов

Замечание

  • Данная информация актуальна только для плагина MultiArc, входящего в стандартную поставку FAR Manager!
ФункцияОписание
CloseArchive закрыть архив
GetArcItem получить информацию о следующем элементе архива
GetDefaultCommands получить строки команд архиватора, используемые по умолчанию
GetFormatName получить имя формата архива
GetSFXPos получить позицию начала архива
IsArchive проверка, является ли указанный файл архивом
LoadFormatModule вызывается во время загрузки плагина второго уровня
OpenArchive открыть архив и подготовить его к чтению
SetFarInfo передает плагину глобальные установки
