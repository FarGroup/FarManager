LStrnicmp

главная | FarStandardFunctions
Функция FSF.LStrnicmp сравнивает части двух строк без учёта регистра. 
int WINAPI LStrnicmp(
  const wchar_t *Str1,
  const wchar_t *Str2,
  int Num
);

Параметры

Str1, Str2
Указатели на строки, которые вы хотите сравнить.
Num
Размер части строки для сравнения.

Возвращаемое значение

Функция возвращает:
-1 - если s1 < s2
1 - если s1 > s2
0 - если s1 == s2
