О проекте Programming FAR plugins - Encyclopedia for Developers v2.0

главная | Лицензия | Setup Help

Цель проекта - создание "полнофункционально-избыточного" файла помощи на русском и английском, применительно для языков программирования C/C++ и Pascal. Данная энциклопедия в основном будет интересна разработчикам плагинов для юникодного FAR Manager версии 2.0.

Энциклопедия существует в двух вариантах - HTMLHelp (pluginsr.chm/pluginse.chm) и OnlineHelp-версия:

pluginsr.chm/pluginse.chm
Применима в составе MSDN (вариант с CHM/CHI-файлами, например, "MSDN Library - July 2001").
pluginsr online
Online-версия энциклопедии. Всегда можно "пролистать" в Сети по адресу: http://api.farmanager.com/

Энциклопедия, кроме оригинальных тем по Plugins API v2.0, содержит множество дополнительных разделов, которые (как мы надеемся) помогут разработчикам плагинов при написании собственных "произведений".
