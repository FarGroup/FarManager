О проекте Programming FAR plugins - Encyclopedia for Developers v2.0

главная | Лицензия | Setup Help

Цель проекта - создание "полнофункционально-избыточного" файла помощи на русском и английском, применительно для языка программирования C/C++.

Данная энциклопедия в основном будет интересна разработчикам плагинов для юникодного FAR Manager версии 2.0.

Энциклопедия существует в двух вариантах - HTMLHelp (CHM) и OnlineHelp-версия.

Энциклопедия, кроме оригинальных тем по Plugins API v2.0, содержит множество дополнительных разделов, которые (как мы надеемся) помогут разработчикам плагинов при написании собственных "произведений".
