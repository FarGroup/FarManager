FAR Manager в Интернете

главная
  • http://www.farmanager.com
    Официальный сайт FAR manager.

  • http://bugs.farmanager.com
    Регистрация и обработка проблем и "хотелок", связанных с FAR Manager и стандартным плагинам к нему.

  • http://api.farmanager.com
    Online версия "Programming FAR plugins - Encyclopedia for Developers"

  • http://forum.farmanager.com
    Официальный форум для пользователей FAR manager.

  • http://plugring.farmanager.com
    Официальный сайт поддержки пользователей оболочки FAR Manager, центральный сайт web-кольца Far PlugRinG - объединенного общества плагинописателей.

  • http://farmanager.com/svn/
    Официальный SVN репозитарий исходных текстов FAR Manager и стандартных плагинов.

  • http://farmanager.com/viewvc/
    Веб интерфэйс к официальному SVN репозитарию.

  • Листы рассылки (с 2007):
    • http://groups.google.com/group/fardev
      Список рассылки "Far Manager Development" предназначен для обсуждения любых вопросов, затрагивающих тему разработки FAR Manager и плагинов к нему.

    • http://groups.google.com/group/farcommits
      Список рассылки "SVN Мониторинг официальных патчей" предназначен для получения информации об изменениях в официальном репозитарии исходных текстов.

  • http://farplugins.sourceforge.net/wiki
    Far Wiki - это место, где собирается информация о Far Manager и его плагинах. Своеобразная народная энциклопедия!

  • news:fido7.far.support
    Конференция USENET FAR.SUPPORT (см. так же на Google Groups)

  • news:fido7.far.development
    Конференция USENET FAR.DEVELOPMENT (см. так же на Google Groups)

  • Листы рассылки (с 2000 по 2007 годы):
    • http://groups.yahoo.com/group/plugring
      В группе рассылки "PlugRinG Forum" обсуждаются плагины к FAR Manager

    • http://groups.yahoo.com/group/farpluginsapi
      Список рассылки "FAR Manager Plugins API" предназначен для общения разработчиков плагинов для FAR Manager и обсуждения вопросов возникших в результате написания плагинов FAR Manager, а так же для любых вопросов, затрагивающих тему "API плагинов для FAR Manager" (в т.ч. обсуждение тем и разделов из "Programming FAR plugins - Encyclopedia for Developers").

    • http://groups.yahoo.com/group/plugring_announce
      Анонсы новых поступление на сайт PlugRinG
