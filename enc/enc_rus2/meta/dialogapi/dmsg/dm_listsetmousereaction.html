DM_LISTSETMOUSEREACTION

Сообщения | Dialog API
Сообщение DM_LISTSETMOUSEREACTION позволяет управлять реакцией на движение мыши в списке DI_LISTBOX.

Param1

ID требуемого элемента диалога.

Param2

Может быть одним из следующих значений (перечисление FARLISTMOUSEREACTIONTYPE):
ЗначениеОписание
LMRT_ONLYFOCUS Список будет реагировать на движение мыши (перемещать курсор в списке) только если элемент в фокусе
LMRT_ALWAYS Список будет всегда реагировать на движение мыши (перемещать курсор в списке). Пример такого поведения - список найденных файлов в результатах поиска (Alt-F7)
LMRT_NEVER Список не будет реагировать на движение мыши

Return

Предыдущее значение.

Элементы

ЭлементОписание
DI_LISTBOX список

Замечания

  1. По умолчанию список DI_LISTBOX всегда реагирует на движение курсора мыши.
