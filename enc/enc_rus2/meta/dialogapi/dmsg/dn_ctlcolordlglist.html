DN_CTLCOLORDLGLIST

События | Dialog API
Сообщение DN_CTLCOLORDLGLIST посылается процедуре обработки диалога перед отрисовкой списка (DI_COMBOBOX, DI_LISTBOX, DIF_HISTORY).

Param1

ID элемента

Param2

Указатель на структуру FarListColors, описывающую цветовую схему списка.

Return

TRUE - если атрибуты изменены
FALSE - оставить все как есть.

Элементы

ЭлементОписание
DI_LISTBOX список
DI_COMBOBOX комбинированный список
DI_EDIT строка ввода (с установленным флагом DIF_HISTORY)
DI_FIXEDIT поле ввода фиксированного размера (с установленным флагом DIF_HISTORY)

Замечания

При обработке этого сообщения процедура обработки диалога может изменить переданные ей атрибуты (цвет текста и фона) для заданного элемента диалога.
