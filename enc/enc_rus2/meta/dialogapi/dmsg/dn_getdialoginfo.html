DN_GETDIALOGINFO

События | Dialog API
Сообщение DN_GETDIALOGINFO посылается в процедуру обработки диалога, FAR желает получить информацию о вашем диалоге.

Param1

0

Param2

Указатель на структуру DialogInfo, получающую данные. Плагин должен корректно заполнить поле StructSize структуры DialogInfo.

Return

Верните TRUE, если заполнили структуру, или FALSE, если проигнорировали.
