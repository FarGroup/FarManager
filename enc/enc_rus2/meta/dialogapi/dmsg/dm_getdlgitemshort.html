DM_GETDLGITEMSHORT

Сообщения | Dialog API
Сообщение DM_GETDLGITEMSHORT посылается Менеджеру диалогов для получения полной информации о заданном элементе. Отличается от DM_GETDLGITEM тем, что FarDialogItem.PtrData всегда возвращается как NULL.

Param1

ID элемента диалога

Param2

Указатель на структуру FarDialogItem.

Return

TRUE - данные скопированы или FALSE - для неверного ID элемента.

Элементы

ЭлементОписание
Все Все элементы диалога

Замечания

Пример

// получить 12-й элемент
// получить параметры 12-го элемента
struct FarDialogItem DialogItem;

Info.SendDlgMessage(hDlg,DM_GETDLGITEMSHORT,12,(LONG_PTR)&DialogItem);
