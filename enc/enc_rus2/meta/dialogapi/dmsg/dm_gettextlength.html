DM_GETTEXTLENGTH

Сообщения | Dialog API
Сообщение DM_GETTEXTLENGTH посылается Менеджеру Диалогов для получения размера данных, содержащихся в FarDialogItem.PtrData.

Param1

ID элемента диалога

Param2

0

Return

Размер данных без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Все элементы диалога.
