DI_COMBOBOX

главная | Dialog API | Элементы диалога
Элемент диалога DI_COMBOBOX описывает поле ввода со списком (комбинированный список). 
struct FarDialogItem
{
  int Type             = DI_COMBOBOX
  int X1               = X1
  int Y1               = Y
  int X2               = X2
  int Y2               = Y (равен значению Y1)
  int Focus            = Focus
  union {
    FarList *ListItems = ListItems  [передаётся]
    int ListPos        = ListPos    [возвращается]
  };
  DWORD Flags          = Flags
  int DefaultButton    = DefaultButton
  const wchar_t *PtrData  = редактируемый текст
};

Внимание!

  • Приведено схематичное изображение структуры; полное описание структуры см. здесь.

Focus

Определяет, имеет ли поле ввода клавиатурный фокус ввода.

ListItems

Указатель на структуру FarList, описывающую пункты для начальной инициализации комбинированного списка. Редактируемое поле заполняется значением первого пункта списка с установленным флагом LIF_SELECTED при условии, что PtrData не содержит "редактируемый текст" и есть хотя бы один элемент списка с таким флагом. Если таких пунктов несколько, берётся первый.

Если значение = NULL, то список выводится не будет.

ListPos

Текущая позиция в списке ListItems.Items. После закрытия диалога это поле содержит позицию выбранного пользователем элемента списка.

Флаги

Для элемента DI_COMBOBOX имеют значения следующие флаги (про флаги списка см. FarListItem):
ФлагОписание
DIF_DROPDOWNLIST стиль нередактируемого раскрывающегося списка - DropDownList
DIF_EDITEXPAND "расширять" переменные среды
DIF_EDITPATH строка ввода будет содержать имена реальных файловых объектов
DIF_LISTAUTOHIGHLIGHT горячие клавиши будут назначены автоматически
DIF_LISTNOAMPERSAND показывать горячие клавиши в списке
DIF_LISTWRAPMODE "заворачивает" список при навигации по списку
DIF_DISABLE элемент является недоступным
DIF_READONLY переводит строки редактирования в состояние "только для чтения"
DIF_SELECTONENTRY выделить редактируемый текст при получении фокуса ввода
DIF_NOFOCUS управляющий элемент не получает клавиатурный фокус ввода
