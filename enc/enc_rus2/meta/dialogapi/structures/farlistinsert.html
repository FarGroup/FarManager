FarListInsert

главная | Dialog API
Структура FarListInsert задаёт параметры для вставки требуемого элемента в списки DI_LISTBOX и DI_COMBOBOX. 
struct FarListInsert
{
  int Index;
  struct FarListItem Item;
};

Элементы

Index
Требуемая позиция элемента в списке. Если Index больше текущего количества элементов списка, то вставляемый пункт будет добавлен в конец списка. Если Index меньше нуля, то вставляемый пункт будет добавлен в начало списка.
Item
Элемент FarListItem.
