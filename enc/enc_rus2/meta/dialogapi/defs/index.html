Макросы

МакросОписание
Dlg_GetDlgData(Info,hDlg) Получить 32-битное значение, связанное с диалогом (DM_GETDLGDATA).
Dlg_GetDlgItemData(Info,hDlg,ID) Получить 32-битное значение, связанное с элементом диалога (DM_GETITEMDATA).
Dlg_RedrawDialog(Info,hDlg) Перерисовать диалоговое окно (DM_REDRAW).
Dlg_SetDlgData(Info,hDlg,Data) Устанавить 32-битное значение связанное с диалогом (DM_SETDLGDATA).
Dlg_SetDlgItemData(Info,hDlg,ID,Data) Устанавить 32-битное значение связанное с элементом диалога (DM_SETITEMDATA).
DlgEdit_AddHistory(Info,hDlg,ID,Str) Добавить строку в историю поля ввода (DM_ADDHISTORY).
DlgItem_Disable(Info,hDlg,ID) Установить элемент диалога в состояние Disable (DM_ENABLE).
DlgItem_Enable(Info,hDlg,ID) Установить элемент диалога в состояние Enable (DM_ENABLE).
DlgItem_GetCheck(Info,hDlg,ID) Получить состояние элементов DI_CHECKBOX и DI_RADIOBUTTON (DM_GETCHECK).
DlgItem_GetFocus(Info,hDlg) Получить ID элемента диалога, имеющего клавиатурный фокус ввода (DM_GETFOCUS).
DlgItem_IsEnable(Info,hDlg,ID) Элемент заблокирован? (DM_ENABLE).
DlgItem_SetCheck(Info,hDlg,ID,State) Установить состояние элементов DI_CHECKBOX и DI_RADIOBUTTON в одно из трех состояний (DM_SETCHECK).
DlgItem_SetFocus(Info,hDlg,ID) Установить клавиатурный фокус на заданный элемент диалога (DM_SETFOCUS).
DlgItem_SetText(Info,hDlg,ID,Str) Установить содержимое строки ввода или заголовка элемента диалога в новое значение (DM_SETTEXTPTR).
DlgList_AddString(Info,hDlg,ID,Str) Добавить строку в список DI_COMBOBOX или DI_LISTBOX (DM_LISTADDSTR).
DlgList_ClearList(Info,hDlg,ID) Очистить список DI_COMBOBOX или DI_LISTBOX (DM_LISTDELETE).
DlgList_DeleteItem(Info,hDlg,ID,Index) Удалить элемент из списка DI_COMBOBOX или DI_LISTBOX (DM_LISTDELETE).
DlgList_GetCurPos(Info,hDlg,ID) Получить позицию курсора в списке DI_COMBOBOX или DI_LISTBOX (DM_LISTGETCURPOS).
DlgList_GetItemData(Info,hDlg,ID,Index) Получить данные, ассоциированные с элементом списка DI_COMBOBOX или DI_LISTBOX (DM_LISTGETDATA).
DlgList_SetCurPos(Info,hDlg,ID,NewPos) Установить новую позицию курсора в списке DI_COMBOBOX или DI_LISTBOX (DM_LISTSETCURPOS).
DlgList_SetItemStrAsData(Info,hDlg,ID,Index,Str) Ассоциировать строку с элементом списка DI_COMBOBOX или DI_LISTBOX (DM_LISTSETDATA).
DlgList_SortDown(Info,hDlg,ID) Сортировать элементы списка DI_COMBOBOX или DI_LISTBOX по убыванию (DM_LISTSORT).
DlgList_SortUp(Info,hDlg,ID) Сортировать элементы списка DI_COMBOBOX или DI_LISTBOX по возрастанию (DM_LISTSORT).
Здесь:
Info
Указатель на структуру PluginStartupInfo.
hDlg
Дескриптор диалога.
ID
Индекс элемента в массиве структур FarDialogItem.
Data
32-битное значение (число типа DWORD или указатель).
Str
Строка.
State
Состояние элемента (одна из констант BSTATE_*, описанная в DM_SETCHECK).
Index
Позиция в списке.
NewPos
Новая позиция в списке.
