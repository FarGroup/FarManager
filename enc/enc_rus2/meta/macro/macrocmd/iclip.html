$IClip

главная | Макросы | Макроязык | Макрокоманды
Оператор $IClip позволяет организовать работу с буфером обмена.

По умолчанию FAR, в операциях запоминания/вставки данных, работает с Буфером Обмена Windows. Макрокоманда '$IClip' позволяет в пределах одного макроса переключать внутренние механизмы FAR для работы с/без использования Буфера Обмена Windows.
Работает как триггер. 

$IClip

Параметры

Нет.

Замечания

  1. Элементы макроязыка невозможно использовать во время записи макроса стандартным образом. Добавить элементы языка в макрос можно только путем ручной правки реестра или путем применения специальных программ/плагинов.

Пример

REGEDIT4

;искать тот же файл на другой панели что и на активной
;F5 ShiftEnter CtrlIns - применено чтобы получить имя файла вне зависимости
;пусто/непусто в cmdline и сколько выделено на панели файлов

[HKEY_CURRENT_USER\Software\Far2\KeyMacros\Shell\CtrlAltLeft]
"Sequence"="$IClip $If (((Bof && APanel.Root) || !Bof) && PPanel.Visible)
$If (fexist(PPanel.Path+\"\\\\\"+APanel.Current) == 1) F5 ShiftEnter
CtrlIns Esc Tab Home Alt< ShiftIns Esc $End $End"
"DisableOutput"=dword:00000001

[HKEY_CURRENT_USER\Software\Far2\KeyMacros\Shell\CtrlAltRight]
"Sequence"="$IClip $If (((Bof && APanel.Root) || !Bof) && PPanel.Visible)
$If (fexist(PPanel.Path+\"\\\\\"+APanel.Current) == 1) F5 ShiftEnter CtrlIns
Esc Tab Home Alt< ShiftIns Esc $End $End"
"DisableOutput"=dword:00000001
