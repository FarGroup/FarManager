Переменные

главная | Макросы | Макроязык
Переменные позволяют вам задавать имена значениям и обращаться к ним позже. В макропоследовательности можно применять локальные и глобальные переменные.

Именование

Имя локальной переменной начинается с символа '%', за которым следуют буквы латинского алфавита, цифры и символ '_'.

Имя глобальной переменной начинается с двух символов '%%', за которым следуют буквы латинского алфавита, цифры, и символ '_'.

Имена регистронезависимые: '%myStr' и '%MYstr' - одна и та же переменная.

Присвоение значения переменной должно заканчиваться символом ';'

Типы

Переменные могут быть трёх типов - целое число, число с плавающей точкой (разделитель целой и дробной частей - точка) или строка.

Целочисленные константы: NNN - 10-ричная константа, 0NNN - 8-ричная константа, 0xNNN - 16-ричная константа.

Целочисленные константы 64-разрядные.

Константы с плавающей точкой - двойной точности 64-разрядные (double).

Область действия

Область действия локальной переменной - текущая макропоследовательность.

Область действия глобальной переменной - текущий сеанс FAR. Глобальные переменные можно записывать в специальную область Vars. Из этой области FAR считывает глобальные переменные при старте.

Представления в реестре

В реестре глобальные переменные располагаются в специальной ветке [HKEY_CURRENT_USER\Software\Far2\[Users\USERNAME\]KeyMacros\Vars].

Каждая глобальная переменная имеет имя и может быть четырёх типов:

  • REG_MULTI_SZ - для строковых переменных
  • REG_SZ - для строковых переменных и чисел с плавающей точкой
  • REG_DWORD - для целочисленных переменных
  • REG_QDWORD - для целочисленных переменных.
Если изначально переменная имела тип REG_DWORD, то при сохранении с помощью функции msave тип меняется на REG_QDWORD.

Преобразование чисел с плавающей точкой в строку осуществляется по формату "KeyMacro/CONVFMT:REG_SZ" (по умолчанию "%.6g")

Замечания

  1. Элементы макроязыка невозможно использовать во время записи макроса стандартным образом. Добавить элементы языка в макрос можно только путём ручной правки реестра или путём применения специальных программ/плагинов.
