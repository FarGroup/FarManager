FarMenuItem

главная | структуры
Структура FarMenuItem описывает один пункт меню. Массив этих структур передается в функцию Menu. 
struct FarMenuItem
{
  const wchar_t *Text;
  int Selected;
  int Checked;
  int Separator;
};

Элементы

Text
Текст пункта меню.
Selected
Флаг активности пункта меню. Только один пункт может быть активным.
Checked
Если не равно нулю, перед текстом отображается метка выбора. Если Checked равно 1, то отображается стандартная метка, если больше 1 - значение Checked используется как символ метки.
Separator
Если не равно нулю, то пункт меню отображается как разделитель. Остальные компоненты структуры (например, Text) игнорируется.
