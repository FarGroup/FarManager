KeySequence

главная | структуры
Структура KeySequence используется в функции AdvControl для передачи последовательности внутренних кодов клавиш в FAR Manager. 
struct KeySequence {
  DWORD Flags;
  int Count;
  DWORD *Sequence;
};

Элементы

Flags
Комбинация флагов (перечисление FARKEYSEQUENCEFLAGS):
ФлагОписание
KSFLAGS_DISABLEOUTPUT Не отображать на экране результат работы каждой клавиши.
KSFLAGS_NOSENDKEYSTOPLUGINS Не передавать клавиши редакторным плагинам (плагинам, экспортирующим функцию ProcessEditorInputW).
Count
Количество кодов клавиш в массиве Sequence.
Sequence
Массив кодов клавиш.
