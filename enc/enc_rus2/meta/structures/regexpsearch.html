RegExpSearch

главная | структуры
Структура RegExpSearch используется в функции RegExpControl Регулярных выражений FAR'а. 
struct RegExpSearch
{
  const wchar_t *Text;
  int Position;
  int Length;
  struct RegExpMatch *Match;
  int Count;
  void *Reserved;
};

Элементы

Text
Исходная строка, в которой будет производиться поиск с использованием Регулярных выражений.
Position
Позиция в строке с которой будет осуществляться поиск.
Length
Длина строки.
Match
Скобки, массив структур RegExpMatch.
Count
Количество скобок.
Reserved
Зарезервировано для будущих нужд, обязательно должено быть NULL.
