ViewerControl

главная | сервисные функции
Функция ViewerControl позволяет запрашивать различную информацию о встроенной программе просмотра FAR Manager и управлять её состоянием. 
int WINAPI ViewerControl(
  int Command,
  void *Param
);

Параметры

Command
Тип команды. Может быть одним из следующих (перечисление VIEWER_CONTROL_COMMANDS):
КомандаОписание
VCTL_GETINFO Получить информацию о программе просмотра. Param указывает на структуру ViewerInfo. Всегда возвращается TRUE.
VCTL_QUIT Закрыть программу просмотра. Param должен быть NULL. Всегда возвращается TRUE.
VCTL_REDRAW Перерисовать окно программы просмотра. Param должен быть NULL. Всегда возвращается TRUE.
VCTL_SETKEYBAR Функция, позволяющая управлять поведением строки функциональных клавиш в программе просмотра:
Param = NULL - восстановить предыдущее значение
Param = -1 - обновить полосу (перерисовать)
Param = указатель на структуру KeyBarTitles.
Всегда возвращается TRUE.
VCTL_SELECT Управление выделением. Param указывает на структуру ViewerSelect.
Если Param = NULL, произойдёт сброс выделения.
В случае успеха возвращается TRUE.
VCTL_SETMODE Изменить режим работы программы просмотра.
Param указывает на структуру ViewerSetMode. В случае успеха возвращается TRUE.
VCTL_SETPOSITION Установить позицию в файле. Param указывает на структуру ViewerSetPosition. В случае успеха возвращается TRUE.

Примечание

VCTL_QUIT в информационной панели и панели быстрого просмотра не приводит к закрытию программы просмотра.
