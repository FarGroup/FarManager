RegExpControl

главная | сервисные функции
Функция RegExpControl позволяет управлять Регулярными выражениями. 
int WINAPI RegExpControl(
  HANDLE hHandle, 
  int Command, 
  LONG_PTR Param  
);

Параметры

hHandle
Смотрите описание Command. (в настоящее время значение игнорируется).
Command
Может быть одним из следующих значений (перечисление FAR_REGEXP_CONTROL_COMMANDS):
КомандаОписание
RECTL_CREATE Создает структуры под Регулярные выражения.
hHandle должен быть установлен в значение NULL.
Param - указатель типа HANDLE, в который запишется созданный хэндл Регулярных выражений.
Функция при неудаче вернет FALSE и Param=INVALID_HANDLE_VALUE.
RECTL_FREE Освобождает память занятую при создании Регулярных выражений.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param игнорируется.
Функция возвращает TRUE.
RECTL_COMPILE Инициализирует Регулярные выражения.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param - указатель на строку с Регулярными выражениями.
Функция возвращает TRUE при удачной компиляции строки.
RECTL_OPTIMIZE Оптимизирует Регуляные выражения после инициализации, применяется после RECTL_COMPILE.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param игнорируется.
Функция возвращает TRUE при удачной оптимизации.
RECTL_MATCHEX Сопоставляет Регуляное выражение строке.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param указатель на заполенную структуру RegExpSearch.
Функция возвращает TRUE при удачной операции сопоставления.
RECTL_SEARCHEX Ищет в строке подстроки заданные Регуляным выражением.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param указатель на заполенную структуру RegExpSearch.
Функция возвращает TRUE, если удалось найти подстроки.
RECTL_BRACKETSCOUNT Количество скобок в Регуляном выражении.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param игнорируется.
Функция возвращает количество скобок в Регуляном выражении.
Param
Указывает на Параметр команды. Смотрите описание Command.

Возвращаемое значение
