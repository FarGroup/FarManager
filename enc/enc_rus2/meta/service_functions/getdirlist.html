GetDirList

главная | сервисные функции
Функция GetDirList возвращает список файлов в указанной директории (с поддиректориями). 
int WINAPI GetDirList(
  const wchar_t *Dir,
  struct FAR_FIND_DATA **pPanelItem,
  int *pItemsNumber
);

Параметры

Dir
Имя обрабатываемого подкаталога текущего каталога. Может быть как именем без пути, так и с полным путем.
pPanelItem
Указывает на переменную, которая получит адрес массива структур FAR_FIND_DATA.
Когда этот массив больше не нужен, он должен быть передан функции FreeDirList.
pItemsNumber
Указывает на переменную, которая получит число структур FAR_FIND_DATA.

Возвращаемое значение

Если функция завершилась успешно, она вернет TRUE. Если функция закончилась неудачно или обработка директории была прервана пользователем, то она вернет FALSE.

Замечания

  1. Функция возвращает полные пути к файлам. Например, если Dir есть D:\DIR1\DIR2, то имена файлов будут в виде D:\DIR1\DIR2\file.ext, D:\DIR1\DIR2\subdir\file.ext и т.п.
  2. Пользователь может прервать выполнение функции, нажав клавишу Esc, при этом функция вернет значение FALSE.
