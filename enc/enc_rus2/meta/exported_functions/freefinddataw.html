FreeFindDataW

главная | экспортируемые функции
Функция FreeFindDataW освобождает память под данные, запрошенную функцией GetFindDataW
(FAR Manager плагину: "я тут списочек запрашивал, так вот, он мне более не нужен, освободи память"). 
void WINAPI FreeFindDataW(
  HANDLE hPlugin,
  struct PluginPanelItem *PanelItem,
  int ItemsNumber
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPluginW или OpenFilePluginW.
PanelItem
Указатель на массив структур типа PluginPanelItem, запрошенных функцией GetFindDataW.
ItemsNumber
Количество элементов в массиве PanelItem.

Возвращаемое значение

Нет.
