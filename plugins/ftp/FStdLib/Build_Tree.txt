Tree
~~~~~~~~~~~~~~
To make FTP from sources I use next directory structure:

 --Directory-----------Files---------------
   <root>              all_far.h
    |                  fstdlib.h
    +--<FARStdlib>     fstd_Multiline.h, funi.h, pack1.h, pack2.h, pack4.h, pop.h
    |                  farcolor.hpp, farkeys.hpp, plugin.hpp
    |
    +--<FarFtp>        ftp_XXX.h
    |                  ftp_XXX.cpp
    +--<Doc>           Doxyfile.fstd   //FStdLib help generation cfg
       |
       +--<html>                       //Autocreated directory for help output html files
 ------------------------------------------

Note: i place "plugin.hpp" file into "FARStdlib" directory because it
      changed a little from it FAR original.

Making
~~~~~~~~~~~~~~
  1. To make plugin you may use
       - Borland BCB builder makefile (*.bcb) in plugin directory
       - Microsoft VC 6.0 makefile (*.vc6) in plugin directory
       - Borland BCB IDE project file (FarFTP.bpr) in plugin directory

  2. Read "Build_ReadMe.txt" file before attempt to build plugin.
