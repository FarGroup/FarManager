This plugin SDK is compatible with Dragon UnPACKer 5.0.0 Beta 1:
Dragon UnPACKer Driver Interface (DUDI) v1

It will not work with older versions (even if DUDI version is the same).
It will work for any future DUP5 supporting DUDI v1.

Included:
---------

drv_none.dpr         Example of a Driver Plugin for DUP5
dup5drv_data.pas     Optional Data handling unit
dup5drv_utils        DUP5 Driver Structures (needed)
