Тут лежат исходники от PCRE 4.4, поправленной кем-то, чтобы компилялась
под Win32. Плюс makefile для Visual C (тестировалось на msvc 7.1). И еще
хедер с оберточными классами для относительно удобного использования
PCRE в C++: называется "pcre++.h" (не путать с библиотекой PCRE++!).
Писал я его для использования в MultiArc'е, поэтому stl там и не пахнет.

В директории test лежит тест этого добра. Test.bat компилирует и
запускает тест. Должно вывести что-то вроде:
